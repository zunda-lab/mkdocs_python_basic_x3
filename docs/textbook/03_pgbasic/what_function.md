# 関数とは

プログラムは、コンピュータへの命令を並べたものです。命令にはいくつかの種類があるんだけど、まずは、命令の代表格である関数を説明します。

## 一連の処理をまとめて実行

関数は、一連の処理をまとめて実行してくれるものです。イメージしやすいように、料理の手順で説明しますね。料理のレシピには「砂糖を大さじ一杯入れる」という手順がありますよね。その一言って、人間の動作で考えると、大さじを取り出して、砂糖が入っている容器の蓋を開け、大さじですりきり一杯すくって、鍋に入れて、容器の蓋を閉めるという一連の作業のことですよね。「砂糖を大さじ一杯入れる」という命令は、そのような一連の処理をまとめて実行するということです。関数も同じで、あれをやって、それをやってをまとめて実行してくれると命令なんです。

## 関数名の後に()が付く

関数には名前が付いています。Pythonのプログラムの中で、アルファベットの名前の後に、丸カッコが付いていれば関数だと思ってください。厳密には、似たようなものに「メソッド」というものがあるんだけど、今の段階では、丸カッコが付いていれば関数だと思ってください。

## 組み込み関数と自作関数

関数には「組み込み関数」と「自作関数」があります。「組み込み関数」はPythonに標準搭載されている関数、「自作関数」は自分で作ったオリジナル関数。Pythonの勉強、はじめのうちは「組み込み関数」だけを使ってプログラムを書いていきましょう。少しプログラムに慣れてきたら、後のミッションで「自作関数」を作ってみますよ。
