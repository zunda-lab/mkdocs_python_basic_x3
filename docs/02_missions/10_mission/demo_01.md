# デモ（1）

それでは、これから「関数を定義する方法」のデモを行います。

「関数を定義しよう」のコーディング演習では、このような画面が表示されます。エディターエリアのコメント文に問題文が転記されていますので、コメント文を読みながら、操作を進めていきましょう。

途中、わからないところがあっても大丈夫、この後のレッスンでしっかり説明します。ご安心ください。

１行目のコメントには「引数を受け取らないシンプルな hello 関数を定義します。」と書かれています。２行目の最初のテンテンテンを削除して「def」と入力します。続けて、後ろのテンテンテンを削除して「丸カッコ・丸カッコ閉じ・コロン」と入力します。

５行目のコメントには「引数をひとつ受け取る good_morning 関数を定義します。」と書かれています。６行目の最初テンテンテンを削除して「def」と入力します。続けて、真ん中のテンテンテンを削除して「丸カッコ」と入力します。続けて、一番後ろのテンテンテンを削除して「丸カッコ閉じ・コロン」と入力します。

９行目のコメントには「hello 関数を呼び出します。」と書かれています。１０行目のテンテンテンを削除して「hello・丸カッコ・丸カッコ閉じ」と入力します。

１１行目のコメントには「good_morning 関数を呼び出します。引数は 3 を指定します。」と書かれています。１２行目のテンテンテンを削除して「good_morning・丸カッコ・3・丸カッコ閉じ」と入力します。

これで完了です。正解かどうか確認しましょう。

「回答を確認する」ボタンをクリックします。

「正解です」が表示されました。

本当に結果が正しく画面に表示されているか確認しましょう。

１３行目に空のprint関数を追加します。１３行目にカーソルを移動して「print・丸カッコ・丸カッコ閉じ」と入力します。

「回答を確認する」ボタンをクリックします。

「答えが間違っています」となりましたが、アウトプットエリアに「こんにちは」と「おはよう」が３回表示されました。プログラムがちゃんと動いている証拠です。

最後に「空のprint関数」を削除します。

「回答を確認する」ボタンをクリックします。

「正解です」に戻りました。
