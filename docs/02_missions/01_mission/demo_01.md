# デモ

それでは、これから、コーディング演習のデモを行います。

コーディング演習のレクチャーに入ると、ご覧のような画面が表示されます。受講生のみなさんは、この画面のままで、コーディング演習を実施してください。私は、みなさんが見やすいように、コーディング演習で使う部分を、全画面表示します。

次に、みなさんが見やすいように、文字のサイズを、４段階、大きくします。

これで、準備ができました。

なお、問題文が半分隠れましたが、このデモでは、私が問題文に書いてあることを説明しながら操作するので大丈夫です。皆さんがトライする場合は、問題文を読んで問題文の通りに進めてください。

Pythonのプログラムが表示されているエリアは、エディターと呼ばれる部分です。

エディターには、print 、カッコ、シングルクォーテーション、Pythonを始めよう、シングルクォーテーション、カッコ閉じ、がはじめから、入力されています。みなさんは、このエリアに、さわらないようにしてください。

エディターの下に、回答を確認するボタンがあります。回答を確認するボタンは、色が付いていないので、一見、ボタンには見えません。ですが、マウスカーソルを持っていくと、背景色が白に変化し、ボタンであることがわかります。

回答を確認するボタンを、クリックします。

ボタンの文言が、確認中に変わります。

「おめでとうございます、正解です！」という文言が表示されましたね。

次に、わざとプログラムを修正して、間違いを体験してみたいと思います。

エディターエリアに、マウスカーソルを移動して、どこでもいいので、一度クリックします。

Pythonの先頭のPを、小文字のpに修正しましょう。

回答を確認するボタンを、クリックします。

ボタンの文言が、確認中に変わります。

答えが間違っています、と表示されました。

答えが間違っています、の下に、どこが間違っているのかが、ヒントとして表示されています。

ヒントの見方は、次のレクチャーで説明しますね。

答えが間違っています、の下の、アウトプットに、pythonを始めよう、が表示されています。プログラムの修正通り、pが小文字となっていますね。

次に、プログラムをリセットすることで、再び、正解です、となることを確認します。

エディターの中の、コードをリセットするボタンをクリックします。

ご覧の通り、確認してください、のダイアログボックスが表示されるので、はい、リセットします、をクリックします。

Pythonの先頭のpが、小文字のpから、大文字のPに戻っていますね。

回答を確認するボタンを、クリックします。

再度、おめでとうございます、正解です！という文言が表示されました。

以上「【デモ】コーディング演習を体験しよう」でした。
