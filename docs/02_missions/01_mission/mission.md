# ミッション

これから、第１ミッションを発表する。

第１ミッションは、（５秒待つ）「コーディング演習を体験せよ」だ。

体験の内容を、もう少し詳しく説明すると、

* 正解ですを体験する
* プログラムの修正方法を体験する
* 間違っていますを体験する

の３点だ。

講師のデモを見て、レッスンにより、知識や手法を十分に理解してから、体験をはじめるので、安心してほしい。

次に、このミッションで、達成できることを紹介する。

このミッションをコンプリートすると、

* コーディング演習を使った、演習の流れがわかる。
* コーディング演習の使い方がわかる。
* 間違ったとき表示される「ヒント」の見方がわかる。
* プログラムがアウトプットする文章や数字の確認方法がわかる。

十分なメリットがあるので、がんばって、コンプリートしてほしい。

以上で、第１ミッション、コーディング演習を体験せよ、の発表を完了する。

検討を祈る。
