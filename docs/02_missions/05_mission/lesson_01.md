## 第５ミッションのコーディング

```
s = 'いぬねこねこいぬねこ'
replaced .....
print(replaced)
```

コーディング演習を始めると、エディタエリアには、スライドに示した半完成プログラムがあらかじめ入力された状態となっています。本当はコメント文も入っていますが、プログラムにフォーカスしてご覧いただくために、コメント文を取り除いて掲載しました。

この半完成プログラムの「テンテンテン」の部分を修正していくんです。

次のスライドで「テンテンテン」に何が入るか説明します。

ネタバレですね。まーいいか。

このスライドでは、エディタエリアにあらかじめ入力されているプログラムとコメント文を説明し、テンテンテンにどのようにプログラムを埋めればいいのかを紹介します。

```
01: s␣=␣'いぬねこねこいぬねこ'
02: # 変数 s に格納されている文字列の "ねこ" を "ニャー" に置換して、変数 replaced に格納します。
03: replaced␣=␣s.replace('ねこ',␣'ニャー')
04: # 置換済みの変数 replaced を表示します。
05: print(replaced)
```


１行目では変数sに文字列「いぬねこねこいぬねこ」を代入しています。

２行目は３行目のコードに対するコメント文です。

３行目には、replacedスペーステンテンテンと入力されていますので、テンテンテンの部分を２行目のコメント「変数 s に格納されている文字列の「ねこ」を「ニャー」に置換して、変数 replaced に格納します。」の通りにコーディングします。赤い字を読み上げます。イコール・スペース・s・ピリオド・replace・丸カッコ・シングルクォーテーション・ねこ・シングルクォーテーション・カンマ・スペース・シングルクォーテーション・ニャー・シングルクォーテーション・丸カッコ閉じ。

これでねこがニャーに置き換わり、変数replacedに代入されます。

４行目は５行目のコードに対するコメント文。

５行目はprint関数でreplaced変数を表示しています。
