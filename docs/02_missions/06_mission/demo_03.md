# デモ（3）

それでは、これから「今日が生まれてから何日目か表示する方法」のデモを行います。

「今日が生まれてから何日目か表示する方法」のコーディング演習では、このような画面が表示されます。エディターエリアのコメント文に問題文が転記されていますので、コメント文を読みながら、操作を進めていきましょう。

途中、わからないところがあっても大丈夫、この後のレッスンでしっかり説明します。ご安心ください。

１行目のコメントには「datetime モジュールの使用を宣言します。」と書かれています。２行目のテンテンテンを削除して「datetime」と入力します。

４行目のコメントには「伊達政宗の誕生日1567年9月5日を変数 birthday に格納します。」と書かれています。５行目のテンテンテンを削除して「イコール・スペース・datetime・ピリオド・date・丸カッコ・1567・カンマ・スペース・9・カンマ・スペース・5・丸カッコ閉じ」と入力します。

６行目のコメントには「今日の日付を取得して変数 today に格納します。」と書かれています。７行目のテンテンテンを削除して「イコール・スペース・datetime・ピリオド・date・ピリオド・today・丸カッコ・丸カッコ閉じ」と入力します。

８行目のコメントには「変数 birthday と today の時刻差を変数 td に格納します。」と書かれています。９行目のテンテンテンを削除して「イコール・スペース・today・スペース・マイナス・スペース・birthday」と入力します。

10行目のコメントには「時刻差が格納された変数 td から差が何日かを取得して変数 days に格納します。」と書かれています。11行目のテンテンテンを削除して「イコール・スペース・td・ピリオド・days」と入力します。

12行目のコメントには「差が何日かが格納された変数 days を表示します。」と書かれています。13行目に書かれているprint関数は修正しません。そのままでOKです。

これで完了です。正解かどうか確認しましょう。

「回答を確認する」ボタンをクリックします。

「正解です」が表示されました。ふ〜。

本当に結果が正しく画面に表示されているか確認しましょう。

14行目に空のprint関数を追加します。14行目にカーソルを移動して「print・丸カッコ・丸カッコ閉じ」と入力します。

「回答を確認する」ボタンをクリックします。

「答えが間違っています」となり、アウトプットエリアに伊達政宗が生まれてから今日までの日数が表示されました。プログラムがちゃんと動いている証拠です。

「先生、この日数、本当に合ってるんですかぁ？」大丈夫です。先生を信じてください！

最後に「空のprint関数」を削除して「回答を確認する」ボタンをクリックします。
