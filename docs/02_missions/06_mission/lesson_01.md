# 今日が生まれてから何日目か表示する方法について学ぼう

第６ミッション、これまでのレクチャーで、今日の日付を取得する方法や日付の引き算について学習してきました。
このレクチャーでは、この後挑むコーディング演習の問題文の説明や、コーディングに関するアドバイスを行います。
さあゴールは目の前です。一緒に学びましょう。

## 第６ミッションの問題文

```
以下の問題文を読み、プログラム中の "....." を修正してください。
伊達政宗が生まれてから今日で何日目かを表示する
1. 2行目の "......" を修正して、datetime モジュールの使用を宣言する文にしてください。
2. 4行目の "....." を修正して、伊達政宗の誕生日1567年9月5日を変数 birthday に格納する文にしてください。日付は datetime.date クラスで作成します。
3. 7行目の "....." を修正して、今日の日付を変数 today に格納する文にしてください。
4. 9行目の "....." を修正して、変数 birthday と today の時刻差を変数 td に格納する文にしてください。
5. 11行目の "....." を修正して、変数 td から差が何日かを取得し変数 days に格納する文にしてください。
```

次のレクチャーはコーディング演習です。このスライドでは、コーディング演習の問題文を紹介し、次のスライドで問題文に対するコーディングアドバイスを紹介します。

7行目のテンテンテンを修正して、今日の日付を変数 today に格納する文にしてください。

## 第６ミッションのコーディング

```
import .....

birthday .....
today .....
td .....
days .....
print(days)
```

コーディング演習を始めると、エディタエリアには、スライドに示した半完成プログラムがあらかじめ入力された状態となっています。本当はコメント文も入っていますが、プログラムにフォーカスしてご覧いただくために、コメント文を取り除いて掲載しました。この半完成プログラムの「テンテンテン」の部分を修正していくんです。

次のスライドで「テンテンテン」に何が入るか説明します。今回もネタバレです。まーいいか。

```
01: # datetime モジュールの使用を宣言します。
02: import␣datetime
03:
04: # 伊達政宗の誕生日1567年9月5日を宣言して変数 birthday に格納します。
05: birthday␣=␣datetime.date(1567,␣9,␣5)
06: # 今日の日付を取得して変数 today に格納します。
07: today␣=␣datetime.date.today()
08: # 変数 birthday と today の時刻差を変数 td に格納します。
09: td␣=␣today␣-␣birthday
10: # 時刻差が格納された変数 td から差が何日かを取得して変数 days に格納します。
11: days␣=␣td.days
12: # 差が何日かが格納された変数 days を表示します。
13: print(days)
```

このスライドでは、エディタエリアにあらかじめ入力されているプログラムとコメント文を説明し、テンテンテンにどのようにプログラムを埋めればいいのかを紹介します。

２行目には「import・スペース・テンテンテン」と入力されていますので、テンテンテンの部分をすぐ上のコメント「datetimeモジュールの使用を宣言します。」の通りにコーディングします。行の先頭から読み上げます。「import・スペース・datetime」

５行目には「birthday・スペース・テンテンテン」と入力されていますので、テンテンテンの部分をすぐ上のコメント「伊達政宗の誕生日1567年9月5日を宣言して変数 birthday に格納します。」の通りにコーディングします。行の先頭から読み上げます。「birthday・スペース・イコール・スペース・datetime・ピリオド・date・丸カッコ・1567・カンマ・スペース・9・カンマ・スペース・5・丸カッコ閉じ」

７行目には「today・スペース・テンテンテン」と入力されていますので、テンテンテンの部分をすぐ上のコメント「 今日の日付を取得して変数 today に格納します。」の通りにコーディングします。行の先頭から読み上げます。「today・スペース・イコール・スペース・datetime・ピリオド・date・ピリオド・today・丸カッコ・丸カッコ閉じ」

９行目には「td・スペース・テンテンテン」と入力されていますので、テンテンテンの部分をすぐ上のコメント「変数 birthday と today の時刻差を変数 td に格納します。」の通りにコーディングします。行の先頭から読み上げます。「td・スペース・イコール・スペース・today・スペース・マイナス・スペース・birthday」

11行目には「days・スペース・テンテンテン」と入力されていますので、テンテンテンの部分をすぐ上のコメント「時刻差が格納された変数 td から差が何日かを取得して変数 days に格納します。」の通りにコーディングします。行の先頭から読み上げます。「days・スペース・イコール・スペース・td・ピリオド・days」

13行目はprint関数でdays変数をディスプレイに表示しています。
