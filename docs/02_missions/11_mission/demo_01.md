# デモ（1）

それでは、これから「ファイルを読み込む方法」のデモを行います。

「ファイルを読み込もう」のコーディング演習では、このような画面が表示されます。エディターエリアのコメント文に問題文が転記されていますので、コメント文を読みながら、操作を進めていきましょう。

途中、わからないところがあっても大丈夫、この後のレッスンでしっかり説明します。ご安心ください。

はじめに、読み込むファイルを確認します。

このコーディング演習には、読み込むファイルを私が準備して仕込んであります。

ファイル名である「fruits.txt」をクリックしてみましょう。

ご覧の通り、５つの果物の名前が格納されたファイルです。このファイルを読み込むプログラムを作ります。

「execise.py」をクリックしてコーディングするファイルに戻りましょう。

１行目と２行目のコメントには「'frutes.txt' ファイルを読み込みモードで開き、戻り値のファイルオブジェクトを変数 f に格納します。」と書かれています。３行目の最初のテンテンテンを削除して「open」と入力します。続けて、後ろのテンテンテンを削除して「シングルクォーテーション・r・シングルクォーテーション」と入力します。

４行目のコメントには「ファイルの内容を読み込み、変数 filedata に格納します。」と書かれています。５行目のテンテンテンを削除して「ピリオド・read・丸カッコ・丸カッコ閉じ」と入力します。

８行目のコメントには「ファイルを閉じます。」と書かれています。９行目のテンテンテンを削除して「ピリオド・close・丸カッコ・丸カッコ閉じ」と入力します。

これで完了です。正解かどうか確認しましょう。

「回答を確認する」ボタンをクリックします。

「正解です」が表示されました。

本当に結果が正しく画面に表示されているか確認しましょう。

１０行目に空のprint関数を追加します。

１０行目にカーソルを移動して「print・丸カッコ・丸カッコ閉じ」と入力します。

「回答を確認する」ボタンをクリックします。

「答えが間違っています」となりましたが、アウトプットエリアに５つの果物の名前が表示されました。プログラムがちゃんと動いている証拠です。

最後に「空のprint関数」を削除します。

「回答を確認する」ボタンをクリックします。

「正解です」に戻りました。

以上「〔デモ〕ファイルを読み込む方法を紹介します」でした。
