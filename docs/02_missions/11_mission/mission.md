# ミッション

これから、第１１ミッションを発表する。

ミッションの内容を、もう少し詳しく説明すると、

* 果物の名前が２０個書かれているファイルを読む
* 「りんご」がいくつかあるか数える
* 「りんご」の数を表示する

の３点だ。

講師がデモをして、レッスンにより知識や手法を十分に理解してから、挑んでもらうので、安心してほしい。

次に、このミッションで、達成できることを紹介する。

このミッションをコンプリートすると

* ファイル入出力の基本手順がわかる
* ファイルオブジェクトとは何かがわかる
* open関数を使ってファイルを開くことができるようになる  
* readメソッドを使ってファイルを読込むことができるようになる
* closeメソッドを使ってファイルを閉じることができるようになる
* countメソッドを使って文字列の出現回数を数えることができるようになる

以上で、第１１ミッション「ファイルを読み指定文字列の出現数をカウントせよ」の発表を完了する。

検討を祈る。
