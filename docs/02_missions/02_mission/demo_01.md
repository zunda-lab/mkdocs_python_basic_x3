# デモ

それでは、これから、第２ミッション「画面に、はじめてのPythonと表示せよ」のデモを行います。第２ミッションのコーディング演習では、このような画面が表示されます。

まず、問題文を読みましょう。問題文には「2行目のテンテンテンの部分を 「はじめてのPython」が表示されるように修正してください。」と書いてあります。エディターの２行目を見てみましょう。「はじめてのPython」の前後に５つのピリオドがありますね。ここを修正しろというわけです。

はじめに前半の５つのピリオドを削除してしまいましょう。削除したら、画面に表示するprint関数を使用するため、関数の名前であるprintを入力します。print関数については、レッスンのレクチャーで説明しますね。printが赤い文字で表示されましたね。Pythonで予約されている単語は、赤い文字で表示されるんです。関数の名前の後は、半角の丸カッコをひとつ入力します。このエディターは、半角の丸カッコを入力すると、閉じる丸カッコも同時に入力されます。プログラミングでは、カッコは対で入力することが多いので、親切に閉じる丸カッコも同時に入力してくれるというわけです。今回は後で閉じカッコを入力しますので、ここでは閉じる丸カッコを削除します。次に、あらかじめ入力してあった「はじめてのPython」という文言の前後にシングルクォーテーションを入力します。「はじめてのPython」が緑色で表示されました。シングルクォーテーションで文言を囲んだことで、文字列として認識されたんです。文字列については、レッスンのレクチャーで説明します。最後に後半の５つのピリオドを削除して、閉じる丸カッコを入力します。

これで完成です。「回答を確認する」ボタンをクリックしてみましょう。「おめでとうございます。正解です！」が表示されました。正解したら「続ける」ボタンを押して次のレクチャーに進んでください。

なお、後のレクチャー「Pythonでプログラムを書くときのルール」で詳しく説明しますが、printと丸カッコ、シングルクォーテーションは必ず半角文字で入力するようにしてください。

丸カッコを全角にするとどうなるか実験してみましょう。初めの丸カッコを全角にして「回答を確認する」ボタンをクリックしてみましょう。「答えが間違っています」と、英語で「invalid charcter（無効な文字）」と表示されました。不注意で全角文字を入力してしまうと、このようにエラーとなりますので、ご注意を。

気持ちが悪いので半角の丸カッコに戻して再度「回答を確認する」ボタンをクリックします。なお、１行目はシャープで始まっていて、文字の色がグレーとなっています。シャープで始まる行には、プログラムの動作に影響しない解説文を書くことができます。コメントと言います。コメントについては、レッスンのレクチャーで説明します。
