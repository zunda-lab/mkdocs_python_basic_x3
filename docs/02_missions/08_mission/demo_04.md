# デモ（4）

それでは、これから「1から100までの3の倍数を表示する方法」のデモを行います。

「1から100までの3の倍数を表示しよう」のコーディング演習では、このような画面が表示されます。エディターエリアのコメント文に問題文が転記されていますので、コメント文を読みながら、操作を進めていきましょう。

途中、わからないところがあっても大丈夫、この後のレッスンでしっかり説明します。ご安心ください。

１行目と２行目のコメントには「１から100の数値を順番に変数 num に格納することを100回繰り返し、num が３の倍数だったら num を表示する」と書かれています。３行目のテンテンテンを削除して「１・カンマ・スペース・101」と入力します。

これで完了です。正解かどうか確認しましょう。

「回答を確認する」ボタンをクリックします。

「正解です」が表示されました。

本当に結果が正しく画面に表示されているか確認しましょう。

６行目に空のprint関数を追加します。６行目にカーソルを移動して「print・丸カッコ・丸カッコ閉じ」と入力します。

「回答を確認する」ボタンをクリックします。

「答えが間違っています」となりましたが、アウトプットエリアには３の倍数「３・６・９・１２」と続き「９３・９６・９９」まで表示されました。プログラムがちゃんと動いている証拠です。

最後に「空のprint関数」を削除します。

「回答を確認する」ボタンをクリックします。

「正解です」に戻りました。

以上「〔デモ〕1から100までの3の倍数を表示する方法を紹介しよう」でした。
