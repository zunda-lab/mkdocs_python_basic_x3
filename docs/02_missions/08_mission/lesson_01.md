# これまで学んだことだけで3の倍数を表示してみよう

第８ミッションのキーワードは「繰り返し」です。

繰り返しはコンピュータのプログラムが最も得意とするところです。私たちは、同じ様なことを１００回やれって言われたら嫌ですよね。しかしプログラムは文句を言わず１００回繰り返してくれる。

このレクチャーでは、Pythonプログラミングの中で繰り返し処理を実行する方法を説明します。

いっしょに学んでいきましょう。

## ３の倍数を表示するには

３の倍数を表示する方法を考えてみましょう。

１を３で割った余りが０か、違う、だから表示しない、２を３で割った余りが０か、違う、だから表示しない、３を３で割った余りが０か、そう０だ、だから表示する、と１から順番に考えていく方法が普通ではないでしょうか？

しかし、この方法で１００まで調べてって言われたら、イヤになってしまうでしょう。

## 同じことを何度も書くことを避ける

プログラミングでは同じことを何度も書くことを避けなければなりません。

例えば同じ処理を１００回実行したいとき、１００回同じ処理を書く方法はダメで、繰り返し実行する処理を書く方法をとります。

プログラミングでは単純作業をいかに減らすかっていう考え方が大切なんです。

この後のレクチャーで繰り返し実行する方法を学んでいきますが、その前に繰り返しを採用しないと、いかに大変かを実感しておきたいと思います。

以上「これまで学んだことだけで3の倍数を表示してみよう」でした。
