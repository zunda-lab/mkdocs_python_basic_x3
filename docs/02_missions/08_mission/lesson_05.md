# 1から100までの3の倍数を表示する方法を学ぼう

第８ミッション、これまでのレクチャーで、繰り返しを行うために、リストやfor文、range関数について学習してきました。

このレクチャーでは、３の倍数を表示するフローチャートをおさらいして、この後挑むコーディング演習の問題文の説明、コーディングに関するアドバイスを行います。

さあゴールは目の前です。一緒に学びましょう。

## ３の倍数を表示するフローチャート

前のレクチャーで「３の倍数を表示するフローチャート」を紹介しましたが、第８ミッション最後のコーディング演習の前にもう一度、３の倍数を表示するためのフローチャートをおさらいします。

まずはfor文の菱形です。

菱形の中のリストはrange関数を使用して準備します。１から１００までの連番ですね。

リストから要素をひとつ取り出し、要素があれば「forブロック」を実行します。

forブロックは１から１００まで１００回実行されます。１０１回目のときに「取り出す要素なし」となり、繰り返しから抜けます。

forブロックの中をズームアップしましょう。

forブロックの中では、if文によって、変数を３で割った余りが０かどうか判断し、０の場合は「Yes」の方向に進みます。０以外のときはNoの方向ですね。Yesの先では、print関数によって変数を表示します。

## 第８ミッションの問題文

次のレクチャーはコーディング演習です。このスライドでは、コーディング演習の問題文を紹介し、次のスライドで問題文に対するコーディングアドバイスを紹介します。

```
以下の問題文を読み、プログラム中の "....." を修正してください。
1から100までの3の倍数を表示する
1. ３行目の "......" を修正して、1 から 100 の数値を順番に変数 num に格納することを 100 回繰り返し、num が 3 の倍数だったら num を表示する文にしてください。
```

問題文を読みます。

以下の問題文を読み、プログラム中のテンテンテンを修正してください。

1から100までの3の倍数を表示する

３行目のテンテンテンを修正して、1 から 100 の数値を順番に変数 num に格納することを 100 回繰り返し、num が 3 の倍数だったら num を表示する文にしてください。

## 第８ミッションのコーディング（1）

コーディング演習を始めると、エディタエリアには、スライドに示した半完成プログラムがあらかじめ入力された状態となっています。

```
for num in range(.....):
    if num % 3 == 0:
        print(num)
```

本当はコメント文も入っていますが、プログラムにフォーカスしてご覧いただくために、コメント文を取り除いて掲載しました。この半完成プログラムの「テンテンテン」の部分を修正していくんです。次のスライドで「テンテンテン」に何が入るか説明します。今回もネタバレっす。

```
01: # 1 から 100 の数値を順番に変数 num に格納することを 100 回繰り返し、
02: # num が 3 の倍数だったら num を表示する
03: for␣num␣in␣range(1,␣101):
04:     if␣num␣%␣3␣==␣0:
05:         print(num)

```

このスライドでは、エディタエリアにあらかじめ入力されているプログラムとコメント文を説明し、テンテンテンにどのようにプログラムを埋めればいいのかを紹介します。

３行目に「for・スペース・num・スペース・in・スペース・range・丸カッコ・テンテンテン・丸カッコ閉じ・コロン」と入力されていますので、テンテンテンの部分をすぐ上のコメント「1 から 100 の数値を順番に変数 num に格納することを 100 回繰り返し、num が 3 の倍数だったら num を表示する」を実現するようにコーディングします。赤い字の部分だけ読み上げます。「1・カンマ・スペース・101」

４行目のif文は変数numを3で割ったあまりがゼロか判断しています。

ゼロの場合は３の倍数なので、５行目でnum変数をprint関数で表示しています。

今回は行数が少なく、穴埋めの箇所も１箇所だけなので、楽勝だったのではないでしょうか？
