# 九九を表示する方法を学ぼう

前のミッションでfor文による繰り返しを学びました。

今回のミッションでは、九九を題材にして、繰り返しの中でさらに繰り返す「二重ループ」という技を体験します。

Pythonでプログラムを作ると「九九」も、それから掛ける数字をひとつ増やした「十十」も、簡単に表示できることを感じていただけたらと思います。

ちなみにインドでは小学校で二桁の九九を習うそうです。私たちはコンピュータに計算してもらいましょう。思い切って「百百」にチャレンジしてみるのも面白いかもしれませんね。

さあ、いっしょに学んでいきましょう。

## 九九の考え方

はじめに九九の考え方をおさらいしましょう。

まずは「１の段」、いんいちがいち、いんにがに、途中省略して、いんくがく。

このスライド作っているとき、この「１の段」という表現がとても懐かしく思いました。「１の段」なんて言ったのは何十年ぶりかなぁ。

つづけて「２の段」、にいちがに、ににがし、途中省略して、にくじゅうはち。

途中の段を省略して、最後「９の段」、くいちがく、くにじゅうはち、途中省略して、くくじゅうはち。

このように、合計８１個の掛け算が九九ですね。

掛けるの記号の前の数字が非乗数、後ろの数字を乗数といいます。このスライドでは非乗数が青い字、乗数は赤い字で示しています。

非乗数が１のとき、つまり１の段では、乗数が１から９まで繰り返されます。

次に２の段、つまり、非乗数が２のとき、同じく乗数は１から９まで繰り返します。

同様に３の段から８の段まで進み、最後、９の段でも、乗数は１から９まで繰り返します。

このように、非乗数がひとつ変化する度、乗数を１から９まで繰り返すということです。

どうでしょう。なんとなく繰り返しの中での繰り返しが見えてきたのではないでしょうか

## 九九のフローチャート

九九の考え方が整理できたので、九九をフローチャートで表してみたいと思います。

まずは、赤い線のループに注目してください。

これは、乗数を１から９まで繰り返すループで、内側のループと言います。

では次に、青い線のループに注目してください。

これは、非乗数を１の段から９の段まで繰り返すループで、外側のループと言います。

外側のループで１の段、２の段、９の段まで回して、段ごとに、内側のループで掛ける数１から９まで繰り返すというわけですね。

２つの菱形は１から９まで繰り返すためのfor文だと思ってください。

どうでしょう。繰り返しの中の繰り返しがはっきりしてきたのではないでしょうか？

これが「二重ループ」と言われるものです。

## 九九の計算結果表示

次に、九九の計算結果をどのように表示するのかを説明します。

画面の大きな四角はディスプレイです。

はじめに青い字に注目してください。

画面の縦方向に１の段から９の段を表示します。非乗数なので外側のループですね。外側の繰り返し１回目で１の段、９回目で９の段が表示されます。

赤い字の内側のループは横方向です。１の段の時の「いんいちがいち」「いんにがに」が横方向に表示されるイメージです。

具体的な動きを、ディスプレイの表示をスローモーションにして解説します。

外側のループ１回目、内側のループ１回目を処理すると「いんいちがいち」の１と空白文字が表示されます。空白文字は次の数字とくっつかないようにするために表示します。この数字と空白文字は、ひとつのprint関数で表示します。print関数をどのように書けば末尾に空白文字が表示されるかは、次のスライドで説明します。

次に、外側のループ１回目、内側のループ２回目を処理すると「いんにがに」の２と空白文字が表示されます。内側のループが次々に処理されると、ご覧のように計算結果と空白文字が次々と表示されるというわけです。

内側のループが９回繰り返された後は、改行文字を表示することで改行します。次の行に次の段を表示するための準備ですね。改行文字の表示もひとつのprint関数で表示します。print関数をどのようにかけば改行文字だけが表示されるは、後で説明します。

外側のループ２回目は２の段です。２の段も同様に内側のループ１回目で、「にいちがに」「ににがし」と続き、最後に改行します。

外側のループが次々と処理され、外側のループ最後は９の段です。

９の段も同様に内側のループ１回目で、「くいちがく」「くにじゅうはち」と続き、最後に改行します。

スローモーションでお伝えしました。実際には、外側のループの中で内側のループがくるくると高速で処理され、一瞬で８１個の数字が表示されます。
