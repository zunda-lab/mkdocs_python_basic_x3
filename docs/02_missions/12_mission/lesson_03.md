# ファイルへ書き込む方法を学ぼう（2）

ファイルに書き込む前に、書き込む内容を文字列として用意する必要があります。

書き込む材料がリストだった場合には、リストを何とかして文字列にしなければなりません。

まずは、このスライドで、リストを結合した文字列を作る方法とはどのようなものかを、皆さんと共有しておきたいと思います。

五つの要素で構成されたリスト型の変数があります。

そしてリストの中には５つの果物の名前「orange・apple・peach・banana・cherry」が格納されています。

このリストを結合して文字列を作ってみましょう。

結合に際して、ひとつ考えなければならないことがあります。それは、間に挟む文字列を指定する必要があるということです。このスライドでは「カンマ」ひとつを間に挟む文字列として指定します。

結合して生成された文字列は「orange・カンマ・apple・カンマ・peach・カンマ・banana・カンマ・cherry」となりました。これは文字列です。果物の名前の間にカンマが挟んでありますね。
