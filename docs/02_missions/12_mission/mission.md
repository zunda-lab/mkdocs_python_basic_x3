# ミッション

これから、第１２ミッションを発表する。

第１２ミッションは「英単語をABC順に並べてファイルに書き込め」だ。

ミッションの内容を、もう少し詳しく説明すると、

* リストに果物の英単語が５個格納されている
* 果物の英単語をABC順に並び替える
* カンマを挟んで連結した文字列を生成する
* 文字列を指定のファイルに書き込む

の４点だ。

講師がデモをして、レッスンにより知識や手法を十分に理解してから、挑んでもらうので、安心してほしい。

次に、このミッションで、達成できることを紹介する。

このミッションをコンプリートすると

* ABC順に並べるsortメソッドが使えるようになる
* リストを結合した文字列を作れるようになる
* writeメソッドを使ってファイルを書き込むことができるようになる

以上で、第１２ミッション「 英単語をABC順に並べてファイルに書き込め」の発表を完了する。
