```

問題文を読みます。

以下の問題文を読み、プログラム中のテンテンテンを修正してください。

ファイルへ書き込む

7行目のテンテンテンを修正して、ファイル「output.txt」を書き込みモードで開き、戻り値のファイルオブジェクトを変数 f に格納する文にしてください。

11行目のテンテンテンを修正して、果物の英単語がリストとして格納された変数 fruits_list の要素をカンマを間にはさんで連結し、連結した文字列を変数 s に格納する文にしてください。

13行目のテンテンテンを修正して、変数 s の文字列をファイルに書き込む文にしてください。

```
fruits_list = ['orange', 'apple', 'peach', 'banana', 'cherry']
filename = 'output.txt'
f = open(filename, .....)
s = ','.....(fruits_list)
f.....(s)
f.close()

```

コーディング演習を始めると、エディタエリアには、スライドに示した半完成プログラムがあらかじめ入力された状態となっています。

本当はコメント文も入っていますが、プログラムにフォーカスしてご覧いただくために、コメント文を取り除いて掲載しました。

この半完成プログラムの「テンテンテン」の部分を修正していくんです。

次のスライドで「テンテンテン」に何が入るか説明します。今回もネタバレです。どんだけー。

```

02: fruits_list␣=␣['orange',␣'apple',␣'peach',␣'banana',␣'cherry']
03: # 書き込むファイルのファイル名を変数 filename に格納します。

06: # 戻り値のファイルオブジェクトを変数 f に格納します。

このスライドでは、エディタエリアにあらかじめ入力されているプログラムとコメント文を説明し、テンテンテンにどのようにプログラムを埋めればいいのかを紹介します。

２行目では、果物の名前を５つの要素としたリストを定義して、変数 fruits_list に代入しています。

４行目は、書き込むファイルのファイル名を、変数 filename に格納しています。

７行目に「f・スペース・イコール・スペース・open・丸カッコ・filename・カンマ・スペース・テンテンテン・丸カッコ閉じ」と入力されていますので、テンテンテンの部分をすぐ上のコメント「ファイル output.txt を書き込みモードで開き、戻り値のファイルオブジェクトを変数 f に格納します。」の通りにコーディングします。赤い字の部分だけ読み上げます。「シングルクォーテーション・w・シングルクォーテーション」

１１行目に「s・スペース・イコール・スペース・シングルクォーテーション・カンマ・シングルクォーテーション・テンテンテン・丸カッコ・fruits_list・丸カッコ閉じ」と入力されていますので、テンテンテンの部分をすぐ上のコメント「リストが格納された変数 fruits_list の要素をカンマを間にはさんで連結し連結した文字列を変数 s に格納します。」の通りにコーディングします。赤い字の部分だけ読み上げます。「ピリオド・join」

１３行目に「f・テンテンテン・丸カッコ・s・丸カッコ閉じ」と入力されていますので、テンテンテンの部分をすぐ上のコメント「変数 s に格納された連結した文字列をファイルに書き込みます。」の通りにコーディングします。赤い字の部分だけ読み上げます。「ピリオド・write」

１５行目で後始末としてファイルをクローズしています。

ちょっと行数が多いのですが、大半はコメント文です。コメント文ではない本当のプログラムの部分に注目すると、意外とシンプルに感じるものです。
