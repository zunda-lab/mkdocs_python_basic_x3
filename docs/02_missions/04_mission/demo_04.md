１１行目のコメントには「変数 base と変数 ope を除算した結果を変数 div に代入します。」と書かれています。１２行目のテンテンテンを削除して「イコール・スペース・base・スペース・スラッシュ・スペース・ope」と入力します。

１３行目のコメントには「変数 base と変数 ope を除算した余りを変数 rem に代入します。」と書かれています。１４行目のテンテンテンを削除して「イコール・スペース・base・スペース・パーセント・スペース・ope」と入力します。
