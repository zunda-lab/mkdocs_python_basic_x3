# print関数とは

## 文字を表示しろという関数

print関数は、ディスプレイに「文字を表示しろ」という関数。ここでの「文字」は数字も含みます。

## 表示した後改行する

print関数は、文字を表示した後、改行します。前のスライドで、関数は一連の処理をまとめて実行すると説明しましたが、print関数は、ディスプレイに文字を表示することと、改行することをまとめて実行してくれる関数なんですね。

## 丸カッコの中で表示文字を指示

表示する文字は、丸カッコの中で指示します。

## print関数の書き方

print関数の書き方を紹介しましょう。

書式はシンプルで、関数名printの後に丸カッコ、丸カッコの中に表示したい文字列や数字を指定し、丸カッコを閉じます。文字列とは何かについては、次のレクチャーで説明しますね。

```
print(表示したい文字列や数字)
```

表示されているプログラム例では、カッコの中で表示する文字列として「はじめてのPython」を指定しています。

``` python
print('はじめてのPython')
```


このprint関数を実行すると、ディスプレイに「はじめてのPython」が表示されるんです。改行付きで。

```
はじめてのPython
（改行）
```
「はじめてのPython」の前後にある「ちょん」と「ちょん」はシングルクォーテーションと言って、文字列を表す記号です。次のレクチャーで詳しく説明しますので、ご安心を。
