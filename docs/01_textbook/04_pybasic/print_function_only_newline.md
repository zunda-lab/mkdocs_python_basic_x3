# print関数で改行だけする方法

このスライドで、print関数で改行だけする方法を説明します。

内側のループを９回まわり終わったあと、次の段を表示する準備のため、改行することを、九九の計算結果表示のスライドでお伝えしました。

print関数は丸カッコの中に何も指定しないと、改行文字だけ表示、つまり、改行だけします。

```
print()
```

プログラム例と実行結果を見てみましょう。

```
print(81,␣end='␣')
print()
```

```
81␣（改行）
```

プログラム例ではprint関数を２行書いています。

１行目のprint関数は、前のスライドで説明した、末尾に空白を表示するprint関数です。このprint関数では改行はせずに空白文字を出力します。

２行目のprint関数は、丸カッコの中に何も指定しないprint関数です。このprint関数が実行されるときに、改行されます。
