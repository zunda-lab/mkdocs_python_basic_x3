# if文（満たさない場合も処理）

次は「満たさない場合も処理」パターンのif文を説明します。

フローチャートで、前のページで紹介した「満たすときときだけ処理」のフローチャートと違うのは、青字のFalse、つまり満たさないときの下に「elseブロック」と呼ばれる箱があることです。

elseとは英語で「でなければ」という意味です。

では書式を説明しましょう。

前のスライドの書式と違うのは、ifブロックの後にelseコロンの行があることです。赤字のコロンは必ず必要です。書かないとエラーになるので気をつけてください。コロンの直後で改行し、満たさない場合に処理したいことをelseブロックとして書きます。elseブロックもifブロック同様、先頭にスペース4つ開けます。

```
if␣num␣%␣2␣==␣0:
␣␣␣␣print('偶数')
else:
␣␣␣␣print('奇数')
```

最後に書式に基づいたプログラム例を書きました。この例はnum変数を2で割った余りが0だったら「偶数」、そうでない場合は「奇数」と表示するものです。

１行目のパーセントは割り算の余りを表すということを四則演算のレッスンで紹介しました。偶数は数字を２で割った余りが０、奇数は２で割った余りが１ですよね。

以上、条件分岐のバリエーション、一つ目と二つ目のif文を紹介しました。三つ目のif文は後のレッスンで紹介します。
