# if文（満たすときだけ処理）

それではいよいよ、if文の書き方を説明します。

前のスライドで分岐条件のバリエーションを三つ紹介しましたが、まず初めは「満たすときだけ処理」パターンのif文を説明します。

スライドでは菱形と箱と矢印で構成された絵をご覧いただいております。実はこれ「フローチャート」と呼ばれ、プログラムの処理の流れをわかりやすく説明する図なんです。

青字で条件式と記載されている菱形は条件を満たすか、そうでないかで矢印が分かれます。

条件を満たすとき、青字でTrueと書かれた矢印の方向に進みます。

そして赤字で「ifブロック」と書かれた処理を実行します。Trueは今のところ「満たす」と読み替えてください。

条件を満たさないときは青字でFalseと書かれた矢印の方向に進み、ifブロックは実行せずに、それ以降の処理に進みます。Falseは今のところ「満たさない」と読み替えてください。

では、この「満たすときだけ処理 if文」の書式を説明しましょう。

初めにifキーワードを記載し、スペースを開けて青字の条件式を記載します。

条件式については、後のスライドで詳しく説明します。条件式の後ろには必ずコロンをひとつ付けて改行します。赤字のコロンは必ず必要です。書かないとエラーになるので気をつけてください。

２行目以降は条件を満たすとき実行する「ifブロック」を書きます。

ifブロックは１行ではなく何行かに渡ることが多いので、ブロックと表記しています。ifブロックは行の先頭に４つのスペースを開けて書きます。このスペース４つについては、後のスライド「ブロックとインデント」で詳しく説明します。

```
if␣num␣%␣2␣==␣0:
␣␣␣␣print('偶数')
else:
␣␣␣␣print('奇数')
```

最後に書式に基づいたプログラム例を書きました。この例は変数scoreが100だったら、満点と表示するものです。100点取ったら満点って言ってもらえる感じですね。

イコールふたつはその左と右が等しいか？という意味で、条件式の一例です。条件式のいくつかの例は、後のスライドで紹介します。
