# オブジェクトとは

オブジェクトとは、コンピュータソフトウェアの言葉で、簡単に言うと「関連するデータと処理をまとめて扱うモノ」です。

英語のobjectは、辞書を引くと、モノや対象という意味で、ソフトウェアの世界では構成する部品ひとつひとつのことを言います。

これまで扱ってきた文字列も正確に言うと「文字列オブジェクト」で、データは格納した文字列、formatメソッドやreplaceメソッドが処理にあたります。データと処理をまとめて扱うモノですよね。

これから扱うファイルオブジェクトもファイルを扱うための部品で、ファイルに格納されてあるデータや、読み書きのためのreadメソッドやwriteメソッドをまとめて扱うことができる部品です。
