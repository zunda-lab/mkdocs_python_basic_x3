# Pythonの特徴

なぜPythonがこんなにも人気なのか、Pythonの３つの特徴で説明しちゃいますね。

一つ目は「文法がシンプル」だということ。

だから、Pythonは、読みやすくて、書きやすくて、少ない行数でプログラムを効率よく書くことができると言われているんです。

シンプルだから、読み書きしやすい。Pythonが短い期間で習得できる、最大の要因と言えるでしょう。

二つ目は「使い勝手がいい命令が豊富」に準備されていること。

Pythonには、簡単ですぐに使える命令が多く標準搭載されています。他のプログラミング言語と比較して、標準搭載されている命令の種類がすごく多いんですよ。

三つ目は「AI・データ分析分野で大活躍」ということ。

多くのエンジニアがPythonを使用してAIのさまざまなジャンルに挑戦しています。

他のプログラミング言語に比べPythonにはAIやデータ分析のプログラミング部品が圧倒的に多い。その部品、とても使い勝手がいいんです。だから、AIやデータ分析のためのプログラムが少ない行数で書けちゃうんです。

以上、「Pythonについて」でした。
