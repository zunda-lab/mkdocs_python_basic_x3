# 二重ループのプログラム（for文）

九九のフローチャートのスライドで「二重ループ」という言葉を紹介しました。

このスライドでは「二重ループ」のプログラムはどのように書くのかを説明します。

まずは書式でイメージをお伝えしましょう。

```
for 変数 in リスト:
    for 変数 in リスト:
        内側のforブロック
```

青い字のfor文をご覧ください。このfor文による繰り返しは外側のループです。九九で言うと１の段、２の段と９の段まで繰り返すループです。

次は赤い字のfor文をご覧ください。赤い字のfor文による繰り返しは内側のループです。九九で言うと１の段の中の掛ける数が１から９まで繰り返すループですね。

その下には内側のforブロックがあって、九九の場合、青い字の変数と赤い字の変数をかけ算した結果を表示します。

「先生、内側のforブロックはわかりましたが、外側のforブロックはないのですか？」そうですよね。そう思いますよね。

実は外側のforブロックは、赤い字のfor文なんです。forブロックにfor文を書くのが「二重ループ」なんです。

次にこの書式に沿ったプログラム例を紹介しましょう。

```
for␣i␣in␣range(1,␣10):
␣␣␣␣for␣j␣in␣range(1,␣10):
␣␣␣␣␣␣␣␣print(i␣*␣j)
```

九九をイメージしたプログラムにしてみました。

い字のfor文は外側のループ、range関数とfor文によって１から９まで繰り返します。

range関数の丸カッコの中、１０を指定している２つ目の数字は、連番最後の数字にプラス１した次の数字なので、９までの繰り返しになります。

内側のループである赤い字のfor文は、外側のforブロックなので、空白文字４つでインデントしてます。こちらもrange関数とfor文によって１から９まで繰り返します。

３行目は内側のforブロックで、外側の変数 i と内側の変数 j を掛けた結果を、print関数によって表示しています。これで九九が表示できます。

変数名が i や j なのは、Pythonに限らず二重ループのプログラミングでは i や j を変数名として使うことが多いからです。本当は非乗数や乗数の英語の方がわかりやすいプログラムだと思いますが、スライドの横幅の字数制限もあり簡単な変数名にしています。

３行目のprint関数は外側のforブロックの中の内側のforブロックなので、空白文字８つでインデントしています。

この九九のプログラムではprint関数での工夫がないので「いんいちがいち」で改行、「いんにがに」で改行と８１行で表示してしまいます。段ごとに改行するプログラムは、この後、紹介します。
