# for文（リスト利用）

それではいよいよ、for文の書き方を説明します。

はじめはフローチャートです。

for文では、繰り返し条件を記述しなくても、自動的にリストの先頭から末尾まで順に繰り返しが行われていきます。取り出す要素があるときはforブロックの処理を実行しますが、取り出す要素がなくなると繰り返しを終えるということです。菱形の中でリストから矢印が変数に向かっていますが、これはリストから取り出した要素を変数に自動的に代入してくれるという意味です。この変数はforブロックの中で使用します。

では、for文の書式を説明しましょう。

```
for 変数 in リスト:
    forブロック
```

初めにforキーワードを記載し、スペースを開けて青字の「変数・スペース・in・スペース・リスト」を記載します。変数の名前については変数名のルールに従っていればなんでも構いません。リストには繰り返し対象のリストが格納された変数名を指定します。リストの後ろには必ずコロンをひとつ付けて改行します。赤字のコロンは必ず必要です。書かないとエラーになるので気をつけてください。

２行目以降はリストから取り出した要素があるとき実行する「forブロック」を書きます。forブロックは１行ではなく何行かに渡ることが多いので、ブロックと表記しています。forブロックはifブロックと同じように行の先頭に４つのスペースを開けて書きます。

最後に書式に基づいたプログラム例を書きました。

```
num_list␣=␣[1,␣2,␣3]
for␣num␣in␣num_list:
    print(num)
```

この例は数字の1、２、３をリストとしてnum_listという変数に代入し、for文でリストであるnum_list変数から要素をひとつずつnumという変数に格納して、print関数でnum変数を表示するというものです。

繰り返しは要素がなくなると終了します。

要素の数分、３回繰り返され、ディスプレイには１、２、３と表示されます。
