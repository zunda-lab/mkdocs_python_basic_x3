# リストを利用した繰り返し

第８ミッションでは、リストを利用した繰り返しを学びます。

リストを利用した繰り返しとは「リストの先頭から末尾までの要素を取り出し、繰り返し何かを実行する」ということです。

「先生、繰り返し何かを実行するってなんですか？」そうですよね。そう思いますよね。

このレクチャーでの「何か」はprint関数です。つまり、ディスプレイに表示することを繰り返すということです。

リスト型のfruits変数で繰り返しをイメージしておきましょう。

fruits変数には４つの文字列「りんご、なし、ぶどう、みかん」が４つの要素として格納されています。この４つの要素を先頭から末尾まで順番にとりだし、ディスプレイに表示するということです。

１回目のループではリストの先頭要素「りんご」を表示します。

２回目のループでは２番目の要素「なし」を表示します。

３回目のループでは３番目の要素「ぶどう」を表示します。

４回目のループではリストの末尾「みかん」を表示します。

この例では４回の繰り返しとなりますね。
