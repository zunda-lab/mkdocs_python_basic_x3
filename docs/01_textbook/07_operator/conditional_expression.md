# 条件式

```
if 条件式:
```

if文の後にくる「条件式」は「分岐する条件」です。

条件式の例を紹介しましょう。

```
age␣==␣20  # 等しい
name␣!=␣'鈴木'  # 等しくない
point␣>␣50  # より大きい
num␣%␣2␣==␣0  # 左辺が式
```

１行目は、変数ageの値が20と等しいかどうか。

２行目は、変数nameの値が文字列「鈴木」と等しくないかどうか。

３行目は、変数pointの値が50より大きいかどうか。

４行目は、変数numの値を２で割って余りが０かどうか。

なんとなく、理解できるのではないでしょうか？

赤い字で示したイコール・イコールやビックリ・イコール、大なり記号は「比較演算子」と言います。比較演算子は「その左と右がどうなのか」という条件の「どうなのか」を表すものですね。

なお、４行目のように比較演算子の左を式で書くこともできます。右も同じです。
