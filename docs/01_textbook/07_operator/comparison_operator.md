# 比較演算子

代表的な比較演算子を紹介しましょう。

| 演算子 | 説明                                |  使用例   |
| :----: | :---------------------------------- | :-------: |
| x == y | x と y は等しいか？（一致）         | age␣==␣20 |
| x != y | x と y は等しくないか？（不一致）   | age␣!=␣20 |
| x > y  | x は y より大きいか？（超過）       | age␣>␣20  |
| x < y  | x は y より小さいか？（未満）       | age␣<␣20  |
| x >= y | x は y より同じか大きいか？（以上） | age␣>=␣20 |
| x <= y | x は y より同じか小さいか？（以下） | age␣<=␣20 |

はじめは「イコール・イコール」です。「x・イコール・イコール・y」と書くと「x と y は等しいか」つまり一致するか？という意味となります。

「先生、数学で等しいはイコールひとつで書きましたぁ」

そうですよね。そう思いますよね。Python プログラミングでは、イコールをひとつ書くと代入、ふたつ書くと等しいという意味になります。数学とは違うのでちょっと頭の切り替えが必要です。例えば「age・スペース・イコール・イコール・スペース・20」と書くと「変数 age の値が 20 と同じか？」という意味となります。年齢が二十歳か？という条件に使えますね。なお、比較演算子の前後のスペース一文字は必須ではないのですが、プログラムが見やすくするため入れるようにしてください。

次は、「ビックリマーク・イコール」です。「x・ビックリマーク・イコール・y」と書くと「x と y は等しくないか？」つまり不一致なのか？という意味となります。例えば「age・スペース・ビックリマーク・イコール・スペース・20」と書くと「変数 age の値が 20 と不一致か？」という意味となります。年齢が二十歳ではないという条件に使えますね。

次は、「大なり記号」です。「x・大なり・y」と書くと「x は y より大きいか？」つまり超過しているか？という意味となります。例えば「age・スペース・大なり・スペース・20」と書くと「変数 age の値が 20 より大きいか？」という意味となります。年齢が二十歳を超えているか？という条件に使えます。

次は、「小なり記号」です。「x・小なり・y」と書くと「x は y より小さいか？」つまり未満か？という意味となります。例えば「age・スペース・小なり・スペース・20」と書くと「変数 age の値が 20 より小さいか？」という意味となります。年齢が二十歳未満か？という条件に使えます。

次は、「大なり・イコール」です。「x・大なり・イコール・y」と書くと「x は y より同じか大きいか？」つまり以上か？という意味となります。例えば「age・スペース・大なり・イコール・スペース・20」と書くと「変数 age の値が 20 と同じか大きいか？」という意味となります。年齢が二十歳以上か？という条件に使えます。

最後は、「小なり・イコール」です。「x・小なり・イコール・y」と書くと「x は y より同じか小さいか？」つまり以下か？という意味となります。例えば「age・スペース・小なり・イコール・スペース・20」と書くと「変数 age の値が 20 と同じか小さいか？」という意味となります。年齢が二十歳以下か？という条件に使えます。

最後にひとつ注意点です。「イコール・イコール」や「大なり・イコール」のように二つの文字で示される比較演算子は間にスペースを入れないようにしてください。スペースを入れるとエラーが発生しまーす。
