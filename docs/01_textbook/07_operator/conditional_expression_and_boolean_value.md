# 条件式と真偽値

「xはyより大きいか」という条件式があったとします。

```
x > y
```

変数xとyの値によって、この条件式を満たすとき、条件式は真、Trueとなります。

逆にこの条件式を満たさないとき、条件式は偽、Falseとなります。

フローチャートで条件式を示す菱形には、Trueの矢印とFalseの矢印２本が出ていたことを思い出してください。どちらの矢印に向かうかを決定するのがこの真か偽かということになります。

また、変数に真偽値を代入することも可能です。

```
flag = False
```

この例の場合は、変数flagにFalseを代入しています。

TrueまたはFalseが代入された変数は「bool型」というデータ型になります。boolは真偽値を表す英語ブーリアンを省略した形です。

フラグの意味や変数に真偽値を入れる意味については、このミッション、最後の方のレッスンで解説します。
