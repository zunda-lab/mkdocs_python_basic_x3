# 文字列を繰り返す

最後に文字列を繰り返す方法を説明します。

前のスライドでちょっとだけ紹介しましたが、改めてお伝えします。

```
'繰り返したい文字列' * 繰り返したい回数
```

文字列を繰り返すには、「アスタリスク演算子」を利用します。「繰り返したい文字列・アスタリスク・繰り返したい回数」と書くと、繰り返された新しい文字列を取得することができます。

プログラム例で紹介しましょう。

```
piyo␣=␣'ぴよ'␣*␣3
print(piyo) 

bar␣=␣'■'␣*␣6
print(bar)
bar␣=␣'■'␣*␣10
print(bar)
```

```
ぴよぴよぴよ（改行）
■■■■■■（改行）
■■■■■■■■■■（改行）
```

このプログラム例では３つの繰り返し文字列を表示しています。

はじめは「ぴよ」という文字列を３回繰り返して、変数piyoに代入し、print関数で表示しています。実行結果に「ぴよぴよぴよ」と表示されていますね。

２つ目の繰り返し文字列は、黒四角ひとつの文字列を、６回繰り返して、変数barに代入し、print関数で表示しています。実行結果の２行目で黒四角が６個横に表示されていますね。

３つ目の繰り返し文字列は、黒四角ひとつの文字列を、１０回繰り返して、変数barに代入し、print関数で表示しています。実行結果の３行目で黒四角が１０個横に表示されていますね。

ひとつ上の行といっしょに眺めると「棒グラフ」に見えませんか？格納したbarと変数名も、棒グラフの「棒」を意味する英単語です。

第１０ミッションは「棒グラフを表示せよ」でしたよね。このテクニックを使って棒グラフを生成します。この後のレッスンでも、作り方をしっかりとお伝えします。
