# 文字列の出現回数を数える

はじめに、文字列の出現回数を数えるとはどのようなことなのかを共有しましょう。

果物の名前を並べたちょっと長い文字列があるとします。

```
'バナナりんごももりんごみかんりんごももバナナりんご'
```

ここに「りんご」という文字列は何回現れるでしょうか？

１回２回３回４回、「りんご」という文字列は４回出現しましたね。

これが文字列の出現回数を数えるということです。

このスライドの例は、ちょっと見れば「りんごは４つ」とわかるので簡単です。

では、１万文字の中にりんごがたくさん埋まっていたとしたらどうでしょう。人間の目で正確に数えるのはしんどいですよね。

プログラムに数えてもらえば、一瞬で正確に数えてくれます。

便利なメソッドなので、ぜひ覚えてほしいメソッドです。

あー、りんごが食べたくなってきましたねー

文字列から指定の文字列の出現回数を数えるには、文字列のcountメソッドを利用します。

```
検索対象の文字列.count(検索する文字列)
```

countメソッドは、検索対象の文字列の後ろに「ピリオド・count・丸カッコ・検索文字列・丸カッコ閉じ」と書きます。

プログラム例を紹介しましょう。

```
s␣=␣'バナナりんごももりんごみかんりんごももバナナりんご'
apple_num␣=␣s.count('りんご')
print(apple_num)
```

```
 4（改行）
```

検索対象の文字列は果物の名前が並んである長い文字列で、変数 s に代入しています。

そして、変数 s のcountメソッドです。

「りんご」という文字列を検索する文字列として引数に指定し、戻り値であるカウント結果を変数 apple_num に代入しています。

最後にprint関数でapple_num変数をディスプレイに表示します。

りんごの数、4が表示されていますね。

以上「文字列の出現回数を数える方法を学ぼう」でした。
