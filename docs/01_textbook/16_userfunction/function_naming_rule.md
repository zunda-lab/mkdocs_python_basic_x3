# 関数の命名ルール

次は、関数の名前を付ける場合のルールをいくつか説明します。第４ミッションで説明した変数の命名ルールと同じですが、あらためて紹介します。

１つ目！英字、数字、アンダースコアのみ使用可能！

アンダースコア以外の記号は使えません。それから正確に言えば、全角文字、日本語も使用できるのですが、推奨されていませんので、特別な事情がない限り使わないようにしてください。

２つ目！先頭、一文字目に数字は使えない！

一文字目は英字またはアンダースコアにする必要があります。

３つ目！関数名の英字は小文字とする慣習あり！

Pythonの関数名は小文字の英字とアンダースコアで命名しましょう。

４つ目！Pythonの予約語は使用禁止！

Pythonには関数名として使用できない「予約語」というものがあります。例えば、class、from、passなどで、35種類あります。全ての予約語は「Python 予約語」をキーワードとしてインターネット検索をすると、見つけることができます。興味があるかたは検索してみてください。
