# 関数とメソッド

ここで関数とメソッドとの違いを説明します。メソッドも関数と同じく、一連の処理の集まりで、呼び出すことで処理をまとめて実行してくれます。似ているので、違いを明らかにしたいと思います。

関数は、基本的に関数名に丸カッコをつけて呼び出します。これまでたくさん使用したprint関数やrange関数などが代表例です。

一方、メソッドは何かというと、文字列や数字などの後ろにピリオドを書き、メソッド名と丸カッコを付けて呼び出します。formatメソッドやreplaceメソッドは、前に文字列があって、ピリオド、メソッド名で使ってきました。

このコースでは、オリジナルのメソッドを作る演習はありません。Pythonに組み込まれているメソッドの使用方法を紹介することにとどめます。

第１０ミッションで扱うのは関数だけ。メソッドの作成方法は、別の機会で紹介できばいいと思っています。
