# オリジナルの関数

作成したオリジナルの関数を使うためには、２つのステップが必要です。

ステップ１は「関数の定義」です。呼び出されたらどのような処理を行うかを書いて、名前を付けます。

ステップ２は「関数の呼び出し」です。関数の名前を書いて関数を呼び出します。呼び出すことで、関数に定義された動作が実行されます。

この２つのステップのことを、それぞれ「関数を定義する」「関数を呼び出す」といいます。

## 引数と戻り値

「関数のイメージ」や「関数のバリエーション」のスライドで少し紹介した「引数」と「戻り値」を改めて紹介します。

関数に仕事をしてもらうには、必要なデータをやりとりしなければならない場合があります。そのようなときに、関数を呼び出す側からデータを送り込むのが「引数」です。呼び出すときに、丸カッコに中に、数値や文字列、リストなどを指定することで、関数が処理をするために必要なデータを、送り込むということです。

「戻り値」は関数から呼び出し元へデータを渡すしくみです。戻り値は変数に代入したり、式の中で計算の材料に使ったり、他の関数の引数にしたりすることができます。

## 関数の定義と呼び出し

先ほど、関数のバリエーションを４つ紹介しました。

ここからは、そのバリエーションごとに、関数の定義の方法と呼び出し方法を説明します。

### 「引数なし・戻り値なし」パターン

はじめは「引数なし・戻り値なし」パターンです。

```
def 関数名():
    処理（関数ブロック）
```

```
def␣hello():  # 関数の定義
␣␣␣␣print('こんにちは')

hello()  # 関数の呼び出し
```

```
こんにちは
```

関数の定義は、「def・スペース・関数名・丸カッコ・丸カッコ閉じ・コロン」から開始します。defは「定義する」を意味する「define」を短縮したものです。

その後の行が関数ブロックで、関数ブロックまでが１つの関数の範囲となります。関数ブロックはこれまで登場してきたifブロックやforブロックと同じく、字下げ（インデント）することで区別されます。

プログラム例を見てみましょう。

このプログラム例はhelloという関数を定義しています。

関数ブロックは１行だけで、print関数で「こんにちは」という文字列を表示しています。たった２行の関数です。

１行、間を空けて、関数を呼び出しています。定義した関数名に丸カッコをつけるだけで呼び出せるんですね。

右のディスプレイをごらんください。定義したオリジナルの関数の呼び出したことで「こんにちは」と表示されています。

### 「引数あり・戻り値なし」パターン

次は、「引数あり・戻り値なし」パターンです。

```
def 関数名(引数1, 引数2, …):
    処理（関数ブロック）
```

```
def␣foo(num):  # 関数の定義
␣␣␣␣print(num)

foo(999)  # 関数の呼び出し
```

```
999
```

引数ありの関数を定義するには、「def・スペース・関数名・丸カッコ・引数１・カンマ・引数２・丸カッコ閉じ・コロン」のような文から開始します。引数は何個でも指定できるので、このような書き方をしているのですが、このコースで作成するオリジナル関数は、引数が一つのものだけです。難しく考えず、引数１だけを定義すると思ってください。

プログラム例を見てみましょう。

このプログラム例はfooという関数を定義しています。丸カッコの中に引数「num」を書いています。引数の名前は変数と同様、好きな名前を付けることができます。

関数ブロックでは、引数で受け取った材料numをprint関数で表示しています。

１行、間を空けて、関数を呼び出しています。foo関数は引数ありなので丸カッコの中で引数に数字の「999」を指定しています。

右のディスプレイをごらんください。引数で指定した「999」が表示されています。

### 「引数なし・戻り値あり」パターン

次は、「引数なし・戻り値あり」パターンです。

```
def 関数名():
    処理（関数ブロック）
    return 戻り値
```

```
def␣bar():  # 関数の定義
␣␣␣␣s␣=␣'Hello World'
␣␣␣␣return␣s

print(bar())  # 関数の呼び出し
```

```
Hello World
```

戻り値ありの関数を定義するには、関数ブロックの最後に「return・スペース・戻り値」を書きます。この戻り値が関数の呼び出し元へ、関数の処理結果として渡されます。

プログラム例を見てみましょう。

このプログラム例はbarという関数を定義しています。

関数ブロックの最後で「return・スペース・s」と書いています。変数 s はその前の行で、文字列「Hello World」を格納していますので、戻り値としては文字列「Hello World」が返却されるというわけです。

１行、間を空けて、関数を呼び出しています。

bar関数は自分でprintしません。呼び出し側のprint関数の丸カッコの中で、bar関数を呼び出し、ディスプレイに表示させています。

右のディスプレイをごらんください。戻り値で返却された「Hello World」が表示されています。

### 「引数あり・戻り値あり」パターン

最後は、「引数あり・戻り値あり」パターンです。

```
def 関数名(引数1, 引数2, …):
    処理（関数ブロック）
    return 戻り値
```

```
def␣baz(count):  # 関数の定義
␣␣␣␣s␣=␣'好き' * count
␣␣␣␣return␣s

print(baz(3))  # 関数の呼び出し
```

```
好き好き好き
```

「引数あり・戻り値あり」パターンは、前の２つのパターンの組み合わせです。

「def・スペース・関数名・丸カッコ・引数１・カンマ・引数２・丸カッコ閉じ・コロン」のような文から開始し、関数ブロックの最後に「return・スペース・戻り値」を書きます。

プログラム例を見てみましょう。

このプログラム例はbazという関数を定義しています。

引数countを受け取り、関数ブロック最後のreturn文で戻り値を返却しています。

returnで返却する変数 s は２行目で生成しています。「好き」という文字列に引数の count を掛けています。文字列に数字を掛けると、繰り返しの文字列を生成することができます。文字列を繰り返す方法は、この後のスライドで詳しく説明しますね。

１行、間を空けて、関数を呼び出しています。baz関数は自分でprintしません。呼び出し側のprint関数の丸カッコの中で、baz関数を呼び出し、ディスプレイに表示させています。baz関数の引数には数値の 3 を指定しています。

右のディスプレイをごらんください。好きが３回表示されています。
