# 繰り返し

３種類のプログラムの流れ「順次」「分岐」「繰り返し」の図、３回目の登場です。

第８ミッションでは、この中の「繰り返し」を学びます。

Pythonプログラミングでは「for文」や「while文」という制御構文を使って繰り返しを行います。

本コースでは「for文」のみ取り扱います。私は多くのPythonプログラムを書いていますが、while文に比べてfor文の利用頻度のほうが圧倒的に多いんです。for文で繰り返しの基礎を学びましょう。

プログラムは上の命令が終了したら、通常、その下の命令を実行しますが、下の命令が終了したら、もう一度同じ命令を実行するという流れです。何回か、繰り返すということですね。

何回繰り返すかは、時と場合によるんですけど、永遠に繰り返すと終わらなくなってしまうので、条件を決めて、その条件を満たしたら終了するようにします。

下の命令は矢印がループになって囲んでいるので「繰り返し」は「ループ」って言ったりもします。
