# プログラムの流れ

スライドに表示された３種類のプログラムの流れ、覚えていますでしょうか？コース冒頭のオリエンテーション、その中の「プログラミングとは」で説明しましたね。

第７ミッションでは、この中の「分岐」を学びます。

Pythonプログラミングでは「if文」という命令を使って分岐を行います。

ちなみに、繰り返しでは「for文」や「while文」という命令。for文は後のミッションで登場します。

if文やfor文などを「制御構文」と言います。

プログラムの流れは基本的に上から下ですが、if文やfor文を使うとプログラムの流れを制御することができるので、プログラミング業界ではこれらを「制御構文」というんです。
