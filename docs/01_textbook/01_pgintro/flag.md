# フラグとは

第７ミッションの問題文に「うるう年フラグ」という言葉が出てきます。

最近は、日常会話でも「フラグが立った」なんて言いますが、このフラグという言葉、プログラミングの世界でも多く使われる言葉なんです。

フラグとは、条件の判定を示す変数です。何とかは何とかか？という条件がYesかNoかなどが入るという意味ですね。

判定結果は、YesやNo、OKやNGなどいろいろな言い方がありますが、プログラミンングはTrueかFalseかという言い方しかしません。フラグにはTrue か Falseが入るんです。そうです。真偽値が入るということですね。

フラグのデータ型はbool型となります。奇数偶数判定のレッスンで少し触れましたが、bool型はTrueかFalseしか入らないデータ型です。boolは真偽値を示す英語ブーリアンの省略形です。

私が小学生のころの話ですが、夏休みのプール開放で、プールの入り口に赤い旗が立っていると、プールに入ることができない決まりになっていました。悪天候などコンディションが悪い状態のとき、当直の方が旗を立てていたんですね。今考えると、あの旗はまさに「フラグ」です。赤い旗が立っていたら、引き返す、立っていなかったらプールに入るという子供の行動を決めていたんですねぇ。

プールとブール、ちょっとわかりずらいかもしれませんね。今の話は水泳するプールの話です。
