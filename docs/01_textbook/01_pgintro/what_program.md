# プログラムとは

コンピュータは、自ら進んで何か行動するものではありません。もし自発的に行動しちゃったら、コンピュータが暴走する映画のように、人類はパニックになっちゃいます。

コンピュータは、人間が「適切な手順」を与えることで、はじめて動くことができる機械です。その「適切な手順」を記述した手順書が「プログラム」というわけです。

プログラムは、コンピュータに対する命令を並べて書いてあるものだと思ってください。

プログラムは別名「コード」とも呼ばれます。

プログラムは、人間が読み書きしやすように、英語に似たイメージで書きますが、1970年頃はコードのイメージに近い数値や記号の羅列でプログラムを書いていたんです。その名残で、今でもコードと呼んでいるんです。
