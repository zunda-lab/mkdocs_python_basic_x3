# ファイルオブジェクト

このスライドでは、Pythonでファイルを扱う場合に重要な存在である「ファイルオブジェクト」について学びます。

「先生、オブジェクトってなんか難しそうなんですけど」

そうですよね。そう思いますよね。オブジェクトについては、次のスライドで学びますので、ちょっと待っててください。

ファイルの内容を変数に読み込んだり、変数の内容をファイルに書き込む場合、Pythonでは変数・ファイル間で直接のデータのやりとりは行いません。

前のスライド「ファイル入出力の基本手順」の手順１「ファイルを開く」をPythonのプログラムの中で行うと、ファイルオブジェクトというものが作成され、指定のファイルとがっちり紐付けがされます。

このファイルオブジェクトを通じて、ファイルの内容を読み込んで変数に格納したり、逆に変数の内容をファイルに書き込んだりします。

「ファイル入出力の基本手順」の手順３「ファイルを閉じる」をPythonのプログラムの中で行うと、このファイルオブジェクトは消滅します。
