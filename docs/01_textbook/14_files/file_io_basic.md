# ファイル入出力の基本手順

Pythonに限らず、プログラムでファイルを扱う場合には、基本、次の手順を踏んで実行します。

* 手順１、ファイルを開く
* 手順２、ファイルを読み込む、または、ファイルを書き込む
* 手順３、ファイルを閉じる

「先生、なんか馴染みがある手順ですね」そうですよね。そう思いますよね。

パソコンでエクセルやパワポなどのファイルを扱うときと同じです。アプリを開くのが先か、ファイルをダブルクリックするのが先かというのはありますが、ファイルを開いて、内容を確認したり、書き込んだりして、最後にファイルを閉じる。アプリを終了すると、同時に、アプリが開いたファイルを閉じているんですよね。

この後のスライドで、この３つの手順をPythonのプログラムから行う方法を紹介します。
