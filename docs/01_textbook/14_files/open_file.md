# ファイルを開く

Pythonでファイルを開くためには、open関数を使用します。

```
ファイルオブジェクト = open(ファイル名, 処理モード)
```

open関数の引数は２つで、１つ目はファイル名、これは文字列で指定します。

２つ目は処理モード、これも文字列で指定します。処理モードに何を指定するのかは、この後、説明します。

open関数の戻り値は、ファイルオブジェクトです。戻り値のファイルオブジェクトは変数に代入します。なぜかと言うと、ファイルを開いた後、ファイルを読み書きするとき、ファイル閉じるときに、戻り値のファイルオブジェクトを使用するためです。

処理モードは何種類かあるのですが、ここでは代表的な４つの処理モードを紹介します。

「r」読み込み専用。

「w」書き込み。ファイルを新規作成します。ファイルがあった場合は上書きします。

「a」も書き込みです。ファイルがあった場合はファイルの最後に追記します。ファイルがなかったら新規作成します。

「r+」読み書き両用です。ファイルを読み込むだけのプログラムの場合、この「r+」読み書き両用でファイルを開くと、プログラムに不具合があった場合、ファイルを壊してしまいますので、用途にあった処理モードを指定するのが賢明です。

また、紹介した４つの処理モードは全てテキストファイルに対しての操作が対象です。画像ファイルなどのバイナリファイルに対しては、違う処理モードで指定します。本コースではテキストファイルへの処理のみ扱います。

プログラム例を紹介します。

```
# ファイル「fruits.txt」を
# 読み込み専用で開いて
# ファイルオブジェクトを変数fに
# 格納する

f␣=␣open('fruits.txt',␣'r')
```

open関数の第１引数に'fruits.txt'という文字列、第２引数に処理モード r を指定しています。

戻り値は変数 f に代入していますね。

このopen関数は、コメントにも書いた通り、ファイル「fruits.txt」を読み込み専用で開いて、ファイルオブジェクトを変数 f に格納するという処理を行います。
