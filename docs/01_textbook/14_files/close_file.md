# ファイルを閉じる

ファイルを閉じるには、ファイルオブジェクトのcloseメソッドを使います。

```
ファイルオブジェクト.close()
```

closeメソッドの引数はありません。戻り値は無いので変数へ代入も不要です。

```
f␣=␣open('fruits.txt',␣'r')

filedata␣=␣f.read()

# ファイルを閉じる
f.close()
```

プログラム例は、コメント文を除くとopen関数とreadメソッド、そしてcloseメソッドの３行です。

openしてreadした後、closeメソッドでファイルを閉じています。

この３行が一番最初のスライドで紹介した「ファイル入出力の基本手順」「手順１ファイルを開く・手順２ファイルを読み込む・手順３ファイルを閉じる」に相当します。

以上「ファイルを読み込む方法を学ぼう」でした。
