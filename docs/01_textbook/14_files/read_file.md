# ファイルの読み込み

## 全部読み

open関数でファイルを開いた後は、ファイルの読み込みです。

ファイルを読み込む方法はいくつかのやり方がありますが、このミッションでは「全部読み」という手法を使います。

スライドでファイルを開いた後の状態をご覧いただいています。

前のスライドのプログラム例の通り、fruits.txt ファイルを開いて、ファイルオブジェクトを生成し、変数 f に格納しました。ファイルとファイルオブジェクトがしっかりと紐付けされています。

fruits.txt ファイルには、５つのくだものの名前が格納されています。「みかん・りんご・もも・バナナ・ぶどう」です。

ファイルに格納された５つの果物の名前を、一つの操作でfiledata変数に入れ込みます。これが全部読みです。

ファイルを読み込む方法には、全部読み以外に、１行ずつ読む方法や、全部読みだけどリストに格納する方法などがありますが、今回は一度に全部読み込んで、文字列として変数に格納する方法を利用します。

```
読み込み結果格納変数 = ファイルオブジェクト.read()
```

ファイルの全部読みをするには、ファイルオブジェクトのreadメソッドを使います。

readメソッドの引数はありません。

戻り値に、ファイルの中身全てが一つの文字列として返ってくるので、読み込み結果を格納する変数に代入します。

```
f␣=␣open('fruits.txt',␣'r')

# ファイルの内容を全て読み込み、変数 filedata に格納する
filedata␣=␣f.read()
```

プログラム例は、コメント文を除くと、open関数とreadメソッドの２行です。

open関数ではファイルオブジェクトを生成して変数 f に代入しています。３行下で、ファイルオブジェクトが格納された変数 f のreadメソッドを利用して「全部読み」を行い、ファイルの中身をfiledataという変数に代入します。

これだけでファイルの全体を一度に読むことができるんです。

簡単ですよね。
