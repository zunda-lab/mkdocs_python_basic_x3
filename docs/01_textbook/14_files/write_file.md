# ファイルへの書き込み

第１１ミッションで「ファイル入出力の基本手順」を説明しました。

「手順１、ファイルを開く」「手順２、ファイルを読み込む、または、ファイルへ書き込む」「手順３、ファイルを閉じる」でしたね。

ファイルへの書き込みも同じ流れです。手順２だけは、読み込みと書き込みでは違うメソッドを使うので、ここでしっかりと説明したいと思います。

メソッドの説明の前に、ファイルに書き込むイメージを合わせておきましょう。

まずは、手順１「ファイルを開く」によって、output.txtファイルを書き込みモードで開くと、output.txtファイルが新規に作成され、ファイルと紐付いたファイルオブジェクト f を生成します。

次に、書き込む文字列を準備します。今回は果物の名前を結合した変数 s を用意しました。変数 s には、改行なしの「orange・カンマ・apple・カンマ・peach・カンマ・banana・カンマ・cherry」という文字列が格納されています。

ファイルオブジェクトに対して「変数 s に格納された文字列を、紐づいたファイルに書き込んでちょうだい」を指令すると、ファイル output.txt に改行なしの「orange・カンマ・apple・カンマ・peach・カンマ・banana・カンマ・cherry」が書き込まれます。

「このファイルに書き込んでちょうだい」と指示する命令が、ファイルオブジェクトのwriteメソッドです。

```
ファイルオブジェクト.write(書き込みたい文字列)
```

ファイルへの書き込みを行うためには、ファイルオブジェクトのwriteメソッドを使います。

writeメソッドの引数には、書き込みたい文字列を指定します。文字列を直接指定するのではなく、文字列が格納された変数を指定することが多いです。

```
f␣=␣open('fruits.txt',␣'w')

# 変数 s に格納された文字列をファイルに書き込む
f.write(s)
```

プログラム例は、コメント文を除くとopen関数とwriteメソッドの２行です。

open関数ではファイルオブジェクトを生成して変数 f に代入しています。このとき open 関数の第２引数「処理モード」は「書き込み（新規作成）」を表す「シングルクォーテーション・w・シングルクォーテーション」を指定します。

最後の行では、生成されたファイルオブジェクトが格納された変数 f のwriteメソッドを書いています。引数には、書き込みたい文字列が格納された s という変数を指定しています。

これだけで指定した文字列をファイルに書き込むことができるんです。簡単ですよね。
