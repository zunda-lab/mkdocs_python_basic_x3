# リストを結合した文字列を作る

リストを結合した文字列を作るには、文字列のjoinメソッドを利用します。

```
結合済文字列 = 間に挟む文字列.join(結合対象のリスト)
```

jonメソッドは、間に挟む文字列の後ろに「ピリオド・join・丸カッコ・結合対象のリスト・丸カッコ閉じ」と書きます。

結合済みの文字列が戻り値として返却されるメソッドです。

プログラム例を紹介しましょう。

```
fruits_list␣=␣['orange',␣'apple',␣'peach',␣'banana',␣'cherry']
s␣=␣','.join(fruits_list)
print(s)
```

```
orange,apple,peach,banana,cherry
```

並び替え対象のリストには、果物の名前「orange・apple・peach・banana・cherry」が５つの要素として格納され、そのリストを変数 fruits_list に代入しています。

そして、joinメソッドです。緑の字の通り、間に挟む文字列としてカンマを指定しています。また、結合対象のリストとしては fruits_list 変数を指定しています。丸カッコの中の青い字ですね。joinメソッドから結合済みの文字列が戻り値として返却されるので、それを変数 s に代入しています。

最後にprint関数で結合した文字列である s 変数をディスプレイに表示しています。

画面では折り返しが入り３行に見えますが、実際は吹き出しの通り、１行の文字列です。

「orange・カンマ・apple・カンマ・peach・カンマ・banana・カンマ・cherry」と表示されています。ABC順への並び替えはまだしていないので、リストに入った順番通りに結合された文字列が表示されています。
