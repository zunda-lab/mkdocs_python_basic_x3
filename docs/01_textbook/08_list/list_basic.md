# リストの基本

## リストとは

これまでは変数には１つの値を代入するものとして説明してきました。１つの数値や文字列などを１つの変数に代入してきましたね。

１つしか入れられないと、たくさんの変数を用意しなければならないので、とても不便です。

そこで、多くのプログラミング言語では１つの変数に複数の値を入れることができる仕組みが準備されています。Pythonでも複数の値を入れられる型がいくつかあるのですが、その代表がリストなんです。

スライドの絵では、fruitsという名前が付いた変数に４つの文字列「りんご、なし、ぶどう、みかん」を格納しています。

## リストの書き方（定義）

それでは、どのように書けばfruits変数をリストとして定義できるのかを紹介しましょう。


```
fruits␣=␣['りんご',␣'なし',␣'ぶどう',␣'みかん'] 
```

fruits変数はご覧のように定義します。

「fruits・スペース・イコール・スペース・角カッコ・シングルクォーテーション・りんご・シングルクォーテーション・カンマ・スペース・シングルクォーテーション・なし・シングルクォーテーション・カンマ・スペース・シングルクォーテーション・ぶどう・シングルクォーテーション・カンマ・スペース・シングルクォーテーション・みかん・シングルクォーテーション・角カッコ閉じ」

これでfruits変数がリストとして定義されます。

スペース文字はあってもなくてもいいのですが、世界中のPythonプログラマーが読みやすいプログラムのために、このような書き方をしていますので、真似して書くようにしてください。

ここでリストに関連する言葉を紹介しましょう。

リストに格納されるデータひとつひとつを「要素」といいます。この例では文字列の「りんご、なし、ぶどう、みかん」が要素ですね。

それから、要素には添字という、0から始まる番号が順番に割り振られていて、この添字を利用して、要素を指定することができます。添字は別名インデックスとも言います。

最後にまとめとして、書式を書いて見ました。

```
変数 = [要素１, 要素２, 要素３, …]
```

ご覧のとおり、代入のイコールの後ろは、角カッコで始まり閉じる角カッコで終わります。そして、その中に要素をカンマ区切りで入れていきます。要素はいくつでも入れられんです。

## リストの参照（取り出し方）

次に、リストからデータを取り出す方法を紹介します。

```
変数名[添字]
```

リスト内の要素を利用するときは、変数名のあとに角カッコで囲んで添字を書きます。

プログラム例で見た方がわかりやすいと思います。

```
print(fruits[0])
print(fruits[1])
print(fruits[2])
print(fruits[3])
```

ご覧のとおり、print関数の丸カッコの中に「fruits・角カッコ・添字・角カッコ閉じ」を入れ込んでいます。４行で０から３までの添字を変化させていますね。

このプログラムを実行すると、ディスプレイに「りんご・なし・ぶどう・みかん」が改行付きで表示されます。

```
りんご（改行）
なし（改行）
ぶどう（改行）
みかん（改行）
```

そんなに難しくないですよね。
