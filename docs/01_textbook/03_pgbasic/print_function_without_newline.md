# print関数で改行しない方法

print関数は文字列や数値をディスプレイに表示した後、改行するということを第２ミッションでお伝えしました。

実は、print関数にちょっとした指示をすることで、改行しないことを指示できるんです。

第７ミッションのコーディング演習では、改行しないprint関数を使いますので、このスライドで学んでおきましょう。

```
print(表示したい文字列や数字, end='')
```

print関数の丸カッコの中で、表示したい文字列や数値を書いた後、「カンマ・スペース・end・イコール・シングルクォーテーション・シングルクォーテーション」を入れると改行しないprint関数となります。print関数は「end・イコール」と文字列が指定されると改行の代わりにその文字列を表示するんです。このスライドの場合は、「end・イコール」の後ろに二つのシングルクォーテーション、つまり、空っぽの文字列を指定していて、改行も、そのほかの文字列も表示しないということになるんです。

プログラム例と実行結果を見てみましょう。

```
print('2200年：',␣end='')
```

```
2200年：
（改行なし）
```

「2200年・コロン」を改行無しで表示したいプログラム例ですね。

第７ミッションのコーディング演習では、このコロンの後に「うるう年です」か「うるう年ではありません」を１行で表示するプログラムを作ります。
