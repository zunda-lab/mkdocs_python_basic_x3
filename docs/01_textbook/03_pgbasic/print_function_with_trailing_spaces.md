# print関数で末尾に空白を表示する方法

このスライドで、print関数で末尾に空白文字を表示する方法を説明します。

前のスライドで説明したとおり、九九の計算結果の表示では、掛け算の結果を表示した後に必ず空白文字を一緒に表示します。

```
print(表示したい文字列や数字, end='␣')
```

print関数の丸カッコの中で、表示したい文字列や数値を書いた後、「カンマ・スペース・end・イコール・シングルクォーテーション・スペース・シングルクォーテーション」を入れると改行の代わりに空白文字を表示します。

print関数は「end・イコール」と文字列が指定されると改行の代わりにその文字列を表示するんです。

このスライドの場合は、「end・イコール」の後ろに空白文字ひとつの文字列を指定しているので、空白文字を出力するというわけです。

「end・イコール」はprint関数のendオプションなどと言ったりしますが、endオプションを指定しない場合は、規定の改行文字が末尾で表示されるんです。だからprint関数は文字や数字を表示した後、改行もするというわけだったんです。明示的にendオプションを指定することで、改行の代わりに指定の文字を表示することができるというわけです。

プログラム例と実行結果を見てみましょう。

```
print(81,␣end='␣')
```

```
81␣（改行しない）
```

「81」と空白文字を表示するプログラム例です。
