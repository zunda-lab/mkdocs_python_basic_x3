# Pythonでプログラムを書くときのルール

プログラムは人間が作りますが、実行するのはコンピュータ。
コンピュータが正確に実行できるように、プログラムにはいろいろなルールがあります。コンピュータは、ルールを守っているプログラムしか、実行してくれないというわけです。
みなさんは日常の生活の中で全てのルールをきっちり守っていますか？交通ルールは守っているけど、家族で決めたルール、守らなかったなんてこと、あるのではないでしょうか？プログラムはルールに厳しいんです。守らないと、うまく動いてくれません。
まだ、プログラミング序の口ですが、このレクチャーでは、最低限守るべきルールについて解説します。

## 全角文字・半角文字のルール

はじめは、全角文字と半角文字に関するルールです。
Pythonのプログラムは、基本的に半角英数字で入力するようにしてください。
スライドのサンプルプログラムを見てください。

```
# メッセージを画面に表示します。
print('はじめてのPython')
```

青字の部分で全角文字を使用しています。引用符で囲まれた文字列の中は全角でも構いません。この例の引用符はシングルクォーテーションですね。また、コメント文も、全角文字オッケーです。
ここで、ワンポイントアドバイス！プログラムを書くときは基本的に日本語入力をオフにするといいでしょう。

## 全角文字・半角文字のルール違反例

全角文字と半角文字のルールについて、ルール違反の例を３つ紹介します。

１つ目！

```
# メッセージを画面に表示します。
ｐｒｉｎｔ('はじめてのPython')
```

関数名 print が全角になっている！実は、これ、ルール違反として紹介しましたが、エラーにならず、実行できてしまうんです。でも、推奨されませんので、関数名は必ず半角文字で書くようにしてください。

２つ目！

```
# メッセージを画面に表示します。
print（'はじめてのPython'）
```

関数名の後の丸カッコが全角！プログラムのどのような種類のカッコも、半角文字のカッコで書くようにしてください。

３つ目！

```
# メッセージを画面に表示します。
print(’はじめてのPython’)
```

引用符が全角文字！文字列を囲むシングルクォーテーション、ダブルクォーテーションは必ず半角文字で書くようにしてください。

## 大文字・小文字のルール

次は大文字・小文字のルールです。
Pythonのプログラムは大文字と小文字を別物として区別します。

```
# メッセージを画面に表示します。
print('はじめてのPython')
```

関数名 print は小文字であると決まっています。

## 大文字・小文字のルール違反例

大文字と小文字のルールについて、ルール違反の例を２つ紹介します。

１つ目！

```
# メッセージを画面に表示します。
PRINT('はじめてのPython')
```

関数名 print が大文字！関数の名前は決められた通りの大文字小文字で書く必要があります。ちなみに、Pythonの関数名は小文字とすることが推奨されています。

２つ目！

```
# メッセージを画面に表示します。
Print('はじめてのPython')
```

関数名 print の先頭文字 P だけ大文字！ふたつのルール違反とも「そんな関数の名前知らないよ」のエラーになってしまいますので、十分に注意しましょう。

# 空白や改行のルール

最後は、空白や改行のルールです。
Pythonのプログラムは空白や改行をしていいところとだめなところがあります。全てを説明すると、それだけで膨大な時間が必要です。最初のうちは余計なエラーで悩まないように、手本をマネて書くようにしてください。

```
#␣メッセージを画面に表示します。↵
print('はじめてのPython')
```

このプログラムでは、コメントの#を後ろにスペースをひとつ入れています。シャープと文字がくっ付くよりは、少し離れていた方がコメント文が読みやすいからです。また、コメント文の後ろで改行をしています。改行しないと print関数もコメント文になってしまいますね。


# 空白や改行のルール違反例

空白と改行のルールについて、ルール違反の例を３つ紹介します。

１つ目！

```
␣print('はじめてのPython')
```

print関数の前に空白１つを入れてしまった！関数名の前にいくつかのスペースを入れることもありますが、今の段階では、行の先頭にはスペースを入れないようにしてください。

２つ目！

```
print('↵
はじめてのPython')
```


文字列を定義する引用符の後ろに改行を入れてしまった！この例の場合の引用符は、シングルクォーテーションですね。引用符で囲って定義する文字列は、１行で定義するルールとなっていますので、この例のように文字列内で改行を入れると、エラーとなってしまいます。

３つ目！

```
print('はじめての↵
Python')
```

文字列の中で改行を入れてしまった！２つ目と同じで、文字列の中に改行を入れると、エラーとなってしまいます。ご注意を。
