    変数を使って四則演算の結果を表示する方法を学ぼう

    第４ミッション、これまでのレクチャーで、四則演算のこと、変数のこと、文字列に変数を埋め込む方法のことを学習してきました。このレクチャーでは、この後挑むコーディング演習の問題文の説明や、コーディングに関するアドバイスを行います。

    「先生、盛りだくさんでお腹いっぱいですー」

    そうですよね。だいぶ盛りだくさんでしたね。もうちょっとしたら第４ミッションも完了です。そうしたらコーヒーでも飲んで休憩しましょう。

    さあゴールは目の前です。一緒に学びましょう。

    第４ミッションの問題文（前半）

    以下の問題文を読み、プログラム中の "....." を修正してください。
変数を定義する
1. 2行目の "......" を修正して、9 を変数 base に代入する文にしてください。
2. 4行目の "......" を修正して、2 を変数 ope に代入する文にしてください。
3. 6行目の "......" を修正して、変数 base と変数 ope を加算した結果を変数 add に代入する文にしてください。
4. 8行目の "......" を修正して、変数 base と変数 ope を減算した結果を変数 sub に代入する文にしてください。
5. 10行目の "......" を修正して、変数 base と変数 ope を乗算した結果を変数 mul に代入する文にしてください。
6. 12行目の "......" を修正して、変数 base と変数 ope を除算した結果を変数 div に代入する文にしてください。
7. 14行目の "......" を修正して、変数 base と変数 ope を除算した余りを変数 rem に代入する文にしてください。

    次のレクチャーはコーディング演習です。このスライドでは、コーディング演習の問題文を紹介します。問題文は長いので、まずは前半部分を紹介し、次のスライドで問題文前半に対するコーディングアドバイス、その後問題文後半とコーディングアドバイスを紹介します。

    問題文（前半）を読みます。

    以下の問題文を読み、プログラム中のテンテンテンを修正してください。

    変数を定義する

    １．2行目のテンテンテンを修正して、9 を変数 base に代入する文にしてください。

    ２．4行目のテンテンテンを修正して、2 を変数 ope に代入する文にしてください。

    ３．6行目のテンテンテンを修正して、変数 base と変数 ope を加算した結果を変数 add に代入する文にしてください。

    ４．8行目のテンテンテンを修正して、変数 base と変数 ope を減算した結果を変数 sub に代入する文にしてください。

    ５．10行目のテンテンテンを修正して、変数 base と変数 ope を乗算した結果を変数 mul に代入する文にしてください。

    ６．12行目のテンテンテンを修正して、変数 base と変数 ope を除算した結果を変数 div に代入する文にしてください。

    ７．14行目のテンテンテンを修正して、変数 base と変数 ope を除算した余りを変数 rem に代入する文にしてください。

    次のスライドでこの問題文に対するコーディングを紹介します。

    第４ミッションのコーディング（前半）

    base .....
ope .....
add .....
sub .....
mul .....
div .....
rem .....

    コーディング演習を始めると、エディタエリアには、スライドに示した半完成プログラムがあらかじめ入力された状態となっています。この半完成プログラムの「テンテンテン」の部分を修正していくんです。

    次のスライドで「テンテンテン」に何が入るか説明します。ネタバレですね。まーいいか。

    このスライドでは、エディタエリアにあらかじめ入力されているプログラムとコメント文を説明し、テンテンテンにどのようにプログラムを埋めればいいのかを紹介します。

    01: # 9 を変数 base に代入します。
02: base␣=␣9
03: # 2 を変数 ope に代入します。
04: ope␣=␣2
05: # 変数 base と変数 ope を加算した結果を変数 add に代入します。
06: add␣=␣base␣+␣ope
07: # 変数 base と変数 ope を減算した結果を変数 sub に代入します。
08: sub␣=␣base␣-␣ope
09: # 変数 base と変数 ope を乗算した結果を変数 mul に代入します。
10: mul␣=␣base␣*␣ope
11:# 変数 base と変数 ope を除算した結果を変数 div に代入します。
12: div␣=␣base␣/␣ope
13:# 変数 base と変数 ope を除算した余りを変数 rem に代入します。
14: rem␣=␣base␣%␣ope

    １行目は２行目のコードに対するコメント文です。

    ２行目には、baseスペーステンテンテンと入力されていますので、テンテンテンの部分を１行目のコメント「 9 を変数 base に代入します。」の通りにコーディングします。赤い字を読み上げます。イコール・スペース・9。グレーはスペース、つまり半角の空白文字です。空白を入れる位置はプログラムを見やすくするため、世界的レベルで慣例化しています。空白の慣例を説明すると短時間ですみませんので、初めのうちは先輩プログラマーを真似して空白を入れようにしてください。次に行きますね。

    ３行目は４行目のコードに対するコメント文です。

    ４行目には、opeスペーステンテンテンと入力されていますので、テンテンテンの部分を３行目のコメント「2 を変数 ope（オペ） に代入します。」の通りにコーディングします。赤い字を読み上げます。イコール・スペース・2。

    ５行目は６行目のコードに対するコメント文です。

    ６行目には、addスペーステンテンテンと入力されていますので、テンテンテンの部分を５行目のコメント「変数 base と変数 ope を加算した結果を変数 add に代入します。」の通りにコーディングします。赤い字を読み上げます。イコール・スペース・base・スペース・プラス・スペース・ope。

    ７行目は８行目のコードに対するコメント文です。

    ８行目には、subスペーステンテンテンと入力されていますので、テンテンテンの部分を７行目のコメント「変数 base と変数 ope を減算した結果を変数 sub に代入します。」の通りにコーディングします。赤い字を読み上げます。イコール・スペース・base・スペース・マイナス・スペース・ope。

    ９行目は１０行目のコードに対するコメント文です。

    １０行目には、mulスペーステンテンテンと入力されていますので、テンテンテンの部分を９行目のコメント「変数 base と変数 ope を乗算した結果を変数 mul に代入します。」の通りにコーディングします。赤い字を読み上げます。イコール・スペース・base・スペース・アスタリスク・スペース・ope。

    １１行目は１２行目のコードに対するコメント文です。

    １２行目には、divスペーステンテンテンと入力されていますので、テンテンテンの部分を１１行目のコメント「変数 base と変数 ope を除算した結果を変数 div に代入します。」の通りにコーディングします。赤い字を読み上げます。イコール・スペース・base・スペース・スラッシュ・スペース・ope。

    １３行目は１４行目のコードに対するコメント文です。

    １４行目には、remスペーステンテンテンと入力されていますので、テンテンテンの部分を１３行目のコメント「変数 base と変数 ope を除算した余りを変数 rem に代入します。」の通りにコーディングします。赤い字を読み上げます。イコール・スペース・base・スペース・パーセント・スペース・ope。
