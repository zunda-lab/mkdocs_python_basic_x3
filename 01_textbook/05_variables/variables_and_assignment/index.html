    変数と代入について学ぼう

    変数とか代入って聞くと数学の方程式を思い出しちゃってなんか難しそうって思っちゃいますよね。

    大丈夫です。Pythonプログラミングの変数や代入は全然難しくありません。

    このレクチャーでは、変数と代入について簡単な例で紹介します。

    変数とは

    このスライドでは、変数とは何かをわかりやすく説明しましょう。

    変数とは、数値や文字列を保存する箱のようなものです。

    変数には名前を付けることができます。この変数の名前を「変数名」と言います。この絵の変数にはyearという変数名が付いています。

    数値を格納してみましょう。変数yearに2020が保存されました。西暦年が2020年だということを意味する変数だと思ってください

    変数に保存する数値や文字列を「値」と言います。値2020をyear変数に入れているんですね。

    このように変数は値を一時的に保存して、あとで取り出して使うための道具なんです。

    代入とは

    このスライドでは、代入について説明します。

    代入とは、変数に値を保存する操作のことです。

    前のスライドに登場した、year変数に2020を保存した例で説明しましょう。

    year = 2020

    Pythonのプログラムでyear変数に数値2020を保存、すなわち代入するには year = 2020 と書きます。これは、左辺の変数yearに右辺の値2020を保存することを意味します。イコールは等しいの意味ではありませんよ。

    このyear = 2020を実行するとき、同じ名前の変数が存在しなければ、Pythonは変数を作成します。このことを「変数を定義する」と言います。もし同じ名前の変数が存在すれば変数の中に入っている値を入れ替えます。元の値が消えて新しい値が入ります。

    これが代入です。

    変数の参照（取り出し）

    値が保存されている変数は、プログラムの中で変数名を書くだけで値を取り出すことができます。値を取り出すことをプログラミングの世界では「参照」と言います。

    実際のプログラム例で説明しましょう。

    year␣=␣2020
print(year)

    サンプルプログラムは２行です。まずは１行目で変数yearに2020を保存しています。その後の２行目では、print関数の丸カッコの中に変数の名前であるyearを指定しています。このprint関数は変数から取り出した値を画面に表示します。2020が表示されていますね。１行目で値が保存されているので、２行目では変数名を書くことでyear変数の値を参照、つまり値を取り出すことができるんですね。

    次のプログラム例は、四則演算を絡めた変数の参照です。

    add␣=␣9␣+␣2　# 9+2の結果を変数addに代入
print(add)

    １行目ではコメント文に書いてある通り、9と2を足し算した結果を変数addに代入しています。つまり、9+2の答えである11が変数addに保存されるっちゅーことですね。

    ２行目のprint関数は変数addを参照しているので、保存されている11が画面に表示されます。図で示した通り変数addを参照して11を取り出しますが、変数に入れた値を参照しても中身は無くなりません。変数の値は残るので、その後も何度を参照することができます。

    add␣=␣9␣+␣2
sub␣=␣9␣-␣2
mul␣=␣9␣*␣2
div␣=␣9␣/␣2
rem␣=␣9␣%␣2

print(add)
print(sub)
print(mul)
print(div)
print(rem)

    おーっと、プログラムが長い。大丈夫ですよ。みなさんならきっと理解できます。一つずつ説明しますね。プログラム例には説明しやすいように行の番号を書きました。行の先頭の二桁の数字が行番号です。その後のコロンとスペースの後ろがプログラムですね。

    １行目では9と2を足した結果を変数addに保存しています。addは足し算の英語であるaddition（アディション）を省略したものです。

    ２行目では9から2を引いた結果を変数subに保存しています。subは引き算の英語であるsubtraction（サブトゥラクション）を省略したものです。

    ３行目では9と2を掛けた結果を変数mulに保存しています。mulは掛け算の英語であるmultiplication（マルティプリケイション）を省略したものです。

    ４行目では9を2で割った結果を変数divに保存しています。divは割り算の英語であるdivision（デヴィジョン）を省略したものです。

    ５行目では9を2で割った余りを変数remに保存しています。remは余りの英語であるremainder（リメインダァ）を省略したものです。

    この５行が実行されると、値が異なる５つの箱が準備されます。それぞれの箱に異なる変数名が付き、異なる値が入っています。

    ７行目から１１行目ではそれぞれの変数を参照してprint関数で画面に表示しています。実行結果のパソコンの画面には、11、9、18、4.5、1が表示されていますね。

    プログラムの行数は多いのですが、落ち着いて考えれば理解できると思います。

    変数の型

    数値や文字列などデータの種類のことを「型」と言います。このスライドでは、変数の型について説明します。 Pythonでは、変数に入る値の種類で、変数の型が決まります。 例えば2020のような数値が保存された変数であれば、変数の型はint型となります。日本語では整数型、intは整数を意味するintegerの略です。 catのような文字列が保存された変数であれば、変数の型はstr型となります。日本語では文字列型、strは文字列を意味するstringの略です。

    前のスライドで、str型とint型を紹介しましたが、Pythonにはいくつもの型があります。このスライドでは代表的な型を紹介します。

    型 意味 例
    str 文字列 'cat', 'はじめてのPython'
    int 整数 2020, -1
    float 小数 4.5, -0.5
    bool 真偽値 True, False

    まずはstr型。日本語では文字列型です。例えば「cat」や「はじめてのPython」などが保存された変数はstr、文字列型です。

    次はint型。日本語では整数型です。これまでは数値と言っていましたが、数値の型は整数型と次に紹介する小数型に分かれます。2020や-1などが保存された変数はint、整数型です。

    次はfloat型。日本語では小数型です。例えば、4.5や-0.5はfloat、小数型です。

    最後はbool型。日本語では真偽値型と言います。Pythonで真を表すTrue、偽を表すFalseがbool、真偽値型です。よくわからないかもしれませんね。真偽値については、第７ミッションで説明するので、今わからなくても、気にしないでください。

    変数の命名ルール

    次は、変数の名前を付ける場合のルールをいくつか説明します。

    １つ目！英字、数字、アンダースコアのみ使用可能！アンダースコア以外の記号は使えません。それから正確に言えば、全角文字、日本語も使用できるのですが、推奨されていませんので、特別な事情がない限り使わないようにしてください。

    ２つ目！先頭、一文字目に数字は使えない！一文字目は英字またはアンダースコアにする必要があります。

    ３つ目！変数名の英字は小文字とする慣習あり！Pythonの変数名は小文字の英字とアンダースコアで命名しましょう。

    ４つ目！Pythonの予約語は使用不可！Pythonには変数名として使用できない「予約語」というものがあります。例えば、class、from、passなどで、35種類あります。全ての予約語は「Python 予約語」をキーワードとしてインターネット検索をすると、見つけることができます。興味があるかたは検索してみてください。

    変数と代入でのエラーあるある

    「変数と代入について学ぼう」最後は初心者のエラーあるあるを紹介します。

    まずは「変数を定義する前に参照してしまった」です。代入をつかって、変数である箱を準備して値を入れる前に、変数から値を取り出そうとするとエラーになってしまいます。

    次は、変数参照時のスペルミスです。代入を使ってyearという名前の変数を用意したのに、参照するときにスペルミスをして「そんな変数しらないよ」的なエラーとなってしまうことです。例えばyear変数を定義したのに、参照するときeとaが逆にしてしまった場合などです。このエラーは初心者に限らず中級者になっても経験するエラーです。変数を参照する場合には、キーボードを手打ちせず、定義した箇所の変数名をコピペするなどの対策をするといいでしょう。

    次は「変数名の先頭文字に数字を使用してしまった」です。変数名の命名ルールのスライドでお話しした通りです。

    最後は「変数名に予約語を使用してしまった」です。気をつけましょう。
